OCR-style practice for placing one selection statement inside another to handle several decisions.

1) Describe the main purpose of Nested Selection constructs in a program. (3 marks)

Mark scheme

Award up to 3 marks:

  • Gives a clear description linked to the topic.
  • Uses correct programming vocabulary.
  • Applies the idea to a suitable example.
  • Accept any other valid point.

2) A cinema checks age first, then checks whether a ticket has been paid for. Write nested selection pseudocode for this. You may use OCR Exam Reference Language or Python. (4 marks)

Mark scheme

Award up to 4 marks:

  • Uses suitable input, output or stored values where relevant.
  • Uses the correct construct or operator for the scenario.
  • Processes data in a logical order.
  • Produces the required result.
  • Accept any other valid point.

OCR-style pseudocode example:

IF age >= 13 THEN
    IF hasTicket = TRUE THEN
        OUTPUT "Enter"
    ENDIF
ENDIF
