2) A ride allows entry if a visitor is at least 13 and has a ticket. Write pseudocode for this decision. You may use OCR Exam Reference Language or Python. (4 marks)

Mark scheme

Award up to 4 marks:

  • Uses suitable input, output or stored values where relevant.
  • Uses the correct construct or operator for the scenario.
  • Processes data in a logical order.
  • Produces the required result.
  • Accept any other valid point.

OCR-style pseudocode example:

IF age >= 13 AND hasTicket = TRUE THEN
    OUTPUT "Enter"
ENDIF
