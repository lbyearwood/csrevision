OCR-style practice for storing several related values in one list and accessing items by position.

1) Describe the main purpose of 1D Lists in a program. (3 marks)

Mark scheme

Award up to 3 marks:

  • Gives a clear description linked to the topic.
  • Uses correct programming vocabulary.
  • Applies the idea to a suitable example.
  • Accept any other valid point.

2) A program stores five lap times in a 1D list. Write pseudocode to output each lap time. You may use OCR Exam Reference Language or Python. (4 marks)

Mark scheme

Award up to 4 marks:

  • Uses suitable input, output or stored values where relevant.
  • Uses the correct construct or operator for the scenario.
  • Processes data in a logical order.
  • Produces the required result.
  • Accept any other valid point.

OCR-style pseudocode example:

scores = [3, 7, 9]
OUTPUT scores[1]
