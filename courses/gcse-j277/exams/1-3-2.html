1.3.2 Star vs Mesh topologies questions

Answer each question, then reveal the hidden mark scheme underneath to check your response.

1) Describe how devices are connected in a star topology. (3 marks)

Mark scheme

1 mark for each correct point:

  • Devices connect to a central device.
  • The central device may be a switch.
  • Data passes through the central device.
  • Each device has its own connection to the central device.
  • Accept any other valid point.

2) Describe how devices are connected in a mesh topology. (3 marks)

Mark scheme

1 mark for each correct point:

  • Devices are connected to many other devices.
  • Data can take more than one route through the network.
  • Some mesh networks connect every device to every other device.
  • A mesh topology can continue working if one connection fails.
  • Accept any other valid point.

3) Explain one advantage of a star topology. (2 marks)

Mark scheme

1 mark for each correct point:

  • If one cable fails, only the connected device is affected.
  • It is easier to add or remove devices.
  • It is easier to identify faults.
  • Performance can be good because each device has its own connection to the central device.
  • Accept any other valid point.

4) Explain one disadvantage of a star topology. (2 marks)

Mark scheme

1 mark for each correct point:

  • If the central device fails, the network may fail.
  • It can require more cable than some other layouts.
  • The central device adds cost.
  • The central device can become a bottleneck.
  • Accept any other valid point.

5) Explain one advantage of a mesh topology. (2 marks)

Mark scheme

1 mark for each correct point:

  • Data can take more than one route.
  • The network can keep working if one connection fails.
  • It can be reliable because there is no single route for all data.
  • It may avoid one central point of failure.
  • Accept any other valid point.

6) Explain one disadvantage of a mesh topology. (2 marks)

Mark scheme

1 mark for each correct point:

  • It can require many connections.
  • It can be expensive to install.
  • It can be harder to set up or manage.
  • More cables or wireless links may be needed.
  • Accept any other valid point.

7) A small office wants a network that is easy to manage and allows devices to be added easily. Recommend a suitable topology and justify your answer. (4 marks)

Mark scheme

1 mark for each correct point:

  • A star topology would be suitable.
  • Devices can be added by connecting them to the central device.
  • It is easier to manage or troubleshoot.
  • If one device connection fails, the other devices can still work.
  • The answer should link the topology to the small office scenario.
  • Accept any other valid point.

8) A network must continue working even if one connection fails. Explain why a mesh topology may be suitable. (3 marks)

Mark scheme

1 mark for each correct point:

  • Mesh networks provide multiple routes for data.
  • If one connection fails, data may travel by another route.
  • This can improve reliability.
  • There is less dependence on one single connection.
  • Accept any other valid point.
