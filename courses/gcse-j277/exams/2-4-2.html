4) A login system allows access when the password is correct and the account is not locked. Create a truth table using True and False. (4 marks)

Mark scheme

  • Includes inputs for password correct and account locked.
  • Includes all four input combinations.
  • Uses NOT account locked, or applies the same rule correctly.
  • Access is True only when password correct is True and account locked is False.
