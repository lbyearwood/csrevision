1.5.1 Operating systems questions

Answer each question, then reveal the hidden mark scheme underneath to check your response.

1) Define what is meant by an operating system. (2 marks)

Mark scheme

1 mark for each correct point:

  • An operating system is system software.
  • It manages the computer’s hardware and software resources.
  • It provides an interface between the user and the computer.
  • Accept any other valid point.

2) Identify three functions of an operating system. (3 marks)

Mark scheme

1 mark for each correct function:

  • User interface.
  • Memory management.
  • Peripheral management.
  • File management.
  • User management.
  • Multitasking.
  • Security.
  • Accept any other valid point.

3) A student uses icons, windows and menus to open files on a laptop. Explain how an operating system provides a user interface. (3 marks)

Mark scheme

1 mark for each correct point:

  • The user interface allows the user to interact with the computer.
  • It may provide a graphical user interface with windows, icons, menus and pointers.
  • It may provide a command-line interface where users type commands.
  • It allows users to open programs, manage files and change settings.
  • Accept any other valid point.

4) Explain how an operating system manages memory. (4 marks)

Mark scheme

1 mark for each correct point:

  • The operating system allocates memory to programs that are running.
  • It keeps track of which areas of memory are in use.
  • It helps prevent programs from interfering with each other’s memory space.
  • It can manage virtual memory when RAM is full.
  • It frees memory when programs are closed.
  • Accept any other valid point.

5) A printer and keyboard are connected to a classroom computer. Explain how an operating system manages peripherals. (3 marks)

Mark scheme

1 mark for each correct point:

  • The operating system controls communication with peripheral devices.
  • It uses device drivers to allow hardware to work with the system.
  • It manages input and output between the computer and devices such as printers, keyboards and monitors.
  • It can queue jobs such as printing tasks.
  • Accept any other valid point.

6) A student saves coursework into folders on a laptop. Explain how an operating system manages files. (4 marks)

Mark scheme

1 mark for each correct point:

  • The operating system organises files and folders.
  • It allows users to create, open, save, move, rename and delete files.
  • It keeps track of where files are stored on secondary storage.
  • It controls permissions for accessing files.
  • It can manage file names, file types and directory structures.
  • Accept any other valid point.

7) Explain how an operating system manages users. (3 marks)

Mark scheme

1 mark for each correct point:

  • The operating system can create and manage user accounts.
  • It can require usernames and passwords for login.
  • It can apply permissions or access levels to different users.
  • It can store user settings and preferences.
  • Accept any other valid point.

8) A student has several programs open at the same time. Explain how the operating system supports multitasking. (4 marks)

Mark scheme

1 mark for each correct point:

  • The operating system allows more than one program to appear to run at the same time.
  • It shares CPU time between running processes.
  • It allocates memory and resources to different programs.
  • It switches between tasks quickly so the user can work with several programs.
  • Accept any other valid point.

9) Explain how an operating system helps keep a computer system secure. (4 marks)

Mark scheme

1 mark for each correct point:

  • It can require users to log in with authentication details.
  • It can control user permissions and access levels.
  • It can manage updates that patch vulnerabilities.
  • It can support encryption or secure storage settings.
  • It can prevent unauthorised users from accessing files or settings.
  • Accept any other valid point.

10) A printer is connected to a computer, but the operating system cannot communicate with it correctly. Explain why a device driver may be needed. (4 marks)

Mark scheme

1 mark for each correct point:

  • A device driver is software that allows the operating system to communicate with hardware.
  • The printer needs instructions in a form it can understand.
  • The driver translates commands between the operating system and the printer.
  • Without the correct driver, the printer may not work properly or may not be recognised.
  • Accept any other valid point.

11) Compare a command-line interface and a graphical user interface. (4 marks)

Mark scheme

1 mark for each correct point:

  • A command-line interface requires users to type commands.
  • A graphical user interface uses visual elements such as windows, icons, menus and pointers.
  • A command-line interface can be efficient for expert users.
  • A graphical user interface is often easier for beginners to use.
  • A command-line interface requires users to remember commands.
  • Accept any other valid point.

12) Explain why a computer needs an operating system to run applications effectively. (6 marks)

Mark scheme

Use the banded mark scheme first, then the indicative content below to decide the final mark.

3-4 Some accurate explanation with relevant points. Covers several relevant points, but the response may lack detail, balance or clear application.
1-2 Limited response with basic understanding. Includes one or two relevant points, but may be brief, generic or partly inaccurate.

Indicative content:

  • The operating system manages hardware resources needed by applications.
  • It allocates CPU time and memory to running programs.
  • It provides file management so applications can open and save data.
  • It manages peripherals used by applications, such as printers and input devices.
  • It provides a user interface so users can start and control applications.
  • It manages security and user permissions.
