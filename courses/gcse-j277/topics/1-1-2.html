1.1.2 CPU performance questions

Answer each question, then reveal the hidden mark scheme underneath to check your response.

1) A student uses a computer to run a single program that performs many calculations. Explain how clock speed can affect CPU performance in this situation. (3 marks)

Mark scheme

1 mark for each correct point:

  • Clock speed is the number of cycles the CPU can carry out per second.
  • A higher clock speed means more fetch-execute cycles can be completed per second.
  • This can allow instructions to be processed more quickly.
  • This could help the calculation program complete tasks more quickly.
  • This can improve overall system performance.
  • A higher clock speed may increase heat output or power use.
  • Accept any other valid point.

2) A school computer is used to run a browser, presentation software and antivirus software at the same time. Explain how the number of cores can affect CPU performance in this situation. (3 marks)

Mark scheme

1 mark for each correct point:

  • A core is a processing unit within the CPU.
  • More cores can allow the CPU to process more than one instruction or task at the same time.
  • This can improve performance when running multiple programs.
  • Different applications or background tasks may be handled by different cores.
  • This can improve performance when using software designed to use multiple cores.
  • More cores will not always improve performance if the software cannot use them effectively.
  • Accept any other valid point.

3) A graphics program repeatedly uses the same image data while applying filters. Explain how cache size can affect CPU performance. (3 marks)

Mark scheme

1 mark for each correct point:

  • Cache stores frequently used data and instructions.
  • Cache is faster to access than RAM.
  • A larger cache can store more frequently used data and instructions.
  • The repeated image data may be fetched from cache instead of RAM.
  • This can reduce the number of times the CPU needs to access RAM.
  • This can improve CPU performance because the CPU spends less time waiting for data.
  • Accept any other valid point.

Mark scheme

1 mark for each correct point:

  • Cache size affects how much frequently used data and instructions can be stored close to the CPU.
  • Larger cache can reduce access to slower main memory.
  • Both can improve CPU performance.
  • Clock speed affects processing rate, whereas cache size affects how quickly data and instructions can be accessed.

Mark scheme

  • A CPU with fewer cores but a higher clock speed or better cache may perform better in some tasks.
