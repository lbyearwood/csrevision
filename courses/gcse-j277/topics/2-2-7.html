2.2.7 Lists / Arrays questions

Practise using one-dimensional and two-dimensional lists or arrays to store and process multiple values.

1) Describe one reason why a programmer may use a list or array. (2 marks)

Mark scheme

Award up to 2 marks:

  • A list or array can store multiple values under one name.
  • It can make repeated processing easier using indexes and loops.
  • Accept any other valid point.

2) A game stores the scores 4, 7 and 9 from three rounds. Write code that stores the scores in a list or array, then outputs the second score. You may use OCR Exam Reference Language or Python. (3 marks)

Mark scheme

Award up to 3 marks:

  • Creates a list or array containing 4, 7 and 9.
  • Accesses the second item using a valid index for the language.
  • Outputs the second score.
  • Accept any other valid point.

OCR-style pseudocode example:

scores = [4, 7, 9]
OUTPUT scores[1]

Python example:

scores = [4, 7, 9]
print(scores[1])

3) A club register stores member names in a list called names. Write code that uses a loop to output every item in the list. You may use OCR Exam Reference Language or Python. (4 marks)

Mark scheme

Award up to 4 marks:

  • Uses a loop.
  • Accesses each item in names.
  • Outputs each item.
  • Stops after every item has been processed.
  • Accept any other valid point.

OCR-style pseudocode example:

FOR index = 0 TO LENGTH(names) - 1
    OUTPUT names[index]
NEXT index

Python example:

for name in names:
    print(name)

4) Explain how a two-dimensional array can be used to represent a table. (3 marks)

Mark scheme

Award up to 3 marks:

  • A 2D array stores data in rows and columns.
  • Each row can represent one record.
  • Each column can represent a field.
  • Accept any other valid point.

5) A teacher gives every student 5 extra marks after a test is moderated. Complete code that adds 5 to every value in a list called marks. You may use OCR Exam Reference Language or Python. (4 marks)

Mark scheme

Award up to 4 marks:

  • Uses a loop to process the list.
  • Accesses each list item.
  • Adds 5 to each value.
  • Stores the updated value back in the list.
  • Accept any other valid point.

OCR-style pseudocode example:

FOR index = 0 TO LENGTH(marks) - 1
    marks[index] = marks[index] + 5
NEXT index

Python example:

for index in range(len(marks)):
    marks[index] = marks[index] + 5
