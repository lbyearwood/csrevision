2.1.10 Trace tables questions

Practise following an algorithm step by step and recording variable values.

1) The algorithm below calculates a running total. Complete the trace table and state the final output. (4 marks)

total = 0
FOR number = 1 TO 4
    total = total + number
NEXT number
OUTPUT total
number total output
1
2
3
4

Mark scheme

Award 1 mark for each correct row of updated values, maximum 4 marks:

number total output
1 1
2 3
3 6
4 10
10 10

Accept any other valid trace that shows the same values in order.

2) The algorithm below uses selection inside a loop. Complete the trace table and state the final output. (5 marks)

score = 0
FOR attempt = 1 TO 3
    INPUT answer
    IF answer == "Y" THEN
        score = score + 2
    ELSE
        score = score - 1
    ENDIF
NEXT attempt
OUTPUT score

The inputs are Y, N, Y.

attempt answer score output
1 Y
2 N
3 Y

Mark scheme

Award 1 mark for each correct score update and 1 mark for the final output, maximum 5 marks:

attempt answer score output
1 Y 2
2 N 1
3 Y 3
3 3

Accept any other valid trace that follows the inputs in the correct order.

values = [4, 7, 2, 9]
largest = values[0]
FOR index = 1 TO 3
    IF values[index] > largest THEN
        largest = values[index]
    ENDIF
NEXT index
OUTPUT largest
index values[index] largest output

Mark scheme

Award 1 mark for each correct largest value after a comparison and 1 mark for the final output, maximum 5 marks:

index values[index] largest output
Start 4

2 2 7
3 9 9
9 9

Accept any other valid trace that shows the same changes to largest.
