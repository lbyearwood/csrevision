1.2.5 Binary number system questions

Answer each question, then reveal the hidden mark scheme underneath to check your response.

1) A sensor in a weather station records a temperature reading before it is processed by a computer. Explain why this data must be converted into binary. (3 marks)

Mark scheme

1 mark for each correct point:

  • Computers use electronic circuits that have two states.
  • Binary uses only two digits: 0 and 1.
  • The temperature reading must be represented using binary values before the computer can process it.
  • Data must be represented as binary so the CPU can process it.
  • All data, including text, images and sound, is stored using binary.
  • Binary allows data to be stored, transferred and processed by a computer system.
  • Accept any other valid point.

2) A music app stores songs, images and instructions for the app. Explain why the computer uses binary to represent this data and these instructions. (3 marks)

Mark scheme

1 mark for each correct point:

  • Computers use electrical signals that can be represented using two states.
  • These two states can be represented as 1 and 0.
  • The songs, images and instructions can all be encoded as patterns of 0s and 1s.
  • Binary is a base-2 number system.
  • Data and instructions must be converted into binary so they can be processed by the computer.
  • Using binary allows computers to store, process and transmit data reliably.
  • Accept any other valid point.

3) Convert the denary number 157 into an 8-bit binary number. Show your working. (3 marks)

Mark scheme

Award up to 3 marks:

  • Correct use of 8-bit place values: 128, 64, 32, 16, 8, 4, 2, 1.
  • Correct selection of place values that total 157: 128 + 16 + 8 + 4 + 1.
  • Correct answer: 10011101.
  • Accept valid working that reaches the correct answer.

4) Convert the binary number 01101011 into denary. Show your working. (3 marks)

Mark scheme

Award up to 3 marks:

  • Correct use of 8-bit place values: 128, 64, 32, 16, 8, 4, 2, 1.
  • Correct selection of place values with a 1: 64 + 32 + 8 + 2 + 1.
  • Correct answer: 107.
  • Accept valid working that reaches the correct answer.

5) State the smallest and largest denary values that can be represented using 8 bits. (2 marks)

Mark scheme

Award up to 2 marks:

  • Smallest value: 0.
  • Largest value: 255.
  • Accept explanation that 8 bits can represent 256 different values, from 0 to 255.

6) Convert the denary number 37 into 8-bit binary. Explain why leading zeroes are needed. (4 marks)

Mark scheme

Award up to 4 marks:

  • Correct use of place values: 128, 64, 32, 16, 8, 4, 2, 1.
  • Correct selection: 32 + 4 + 1.
  • Correct answer: 00100101.
  • Leading zeroes are used so the value is shown as a full 8-bit binary number.

7) The binary number 10000000 has a 1 in the most significant bit. Explain what this means in an 8-bit unsigned binary number. (3 marks)

Mark scheme

1 mark for each correct point:

  • The most significant bit is the leftmost bit.
  • In an 8-bit unsigned binary number, the leftmost bit has the place value 128.
  • 10000000 represents the denary value 128.
  • The least significant bit is the rightmost bit and has the place value 1.
