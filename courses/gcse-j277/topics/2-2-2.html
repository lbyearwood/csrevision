2.2.2 Data types questions

Practise choosing and using suitable data types, including integer, real, Boolean, character, string and casting.

1) A school login system stores age, surname, exam percentage and whether the user is logged in. Identify a suitable data type for each item. (4 marks)

Mark scheme

Award up to 4 marks:

  • Age: integer.
  • Surname: string.
  • Exam percentage: real or integer depending on whether decimals are needed.
  • Logged-in status: Boolean.
  • Accept any other valid point.

2) Explain why using suitable data types is important when writing a program. (3 marks)

Mark scheme

Award up to 3 marks:

  • The data type controls what values can be stored.
  • The data type affects what operations can be performed.
  • Using a suitable data type can reduce errors or make the program clearer.
  • Accept any other valid point.

3) A quiz app receives a score from a text box. Write code that inputs the score as text, casts it to an integer, doubles it for a bonus round and outputs the result. You may use OCR Exam Reference Language or Python. (4 marks)

Mark scheme

Award up to 4 marks:

  • Inputs or stores the text value.
  • Casts or converts the value to an integer.
  • Doubles the integer value.
  • Outputs the result.
  • Accept any other valid point.

OCR-style pseudocode example:

numberText = INPUT
number = INTEGER(numberText)
answer = number * 2
OUTPUT answer

Python example:

number_text = input()
number = int(number_text)
answer = number * 2
print(answer)

4) Describe the difference between an integer and a real data type. (2 marks)

Mark scheme

Award up to 2 marks:

  • An integer stores a whole number.
  • A real stores a number that can include a decimal part.
  • Accept any other valid point.

5) A driving theory app needs to store whether a learner has passed a mock test. Complete code that stores this in a Boolean variable called passed, then outputs the value. You may use OCR Exam Reference Language or Python. (3 marks)

Mark scheme

Award up to 3 marks:

  • Uses a Boolean value.
  • Assigns the Boolean value to passed.
  • Outputs passed.
  • Accept any other valid point.

OCR-style pseudocode example:

passed = TRUE
OUTPUT passed

Python example:

passed = True
print(passed)
