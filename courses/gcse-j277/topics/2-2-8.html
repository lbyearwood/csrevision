2.2.8 Sub programs questions

Practise functions and procedures, including parameters, return values and local and global variables.

1) Compare a function and a procedure. (4 marks)

Mark scheme

Award up to 4 marks:

  • A function returns a value.
  • A procedure performs a task but does not have to return a value.
  • Both are sub programs.
  • Both can help structure a program and reduce repeated code.
  • Accept any other valid point.

2) Explain one benefit of using sub programs in a larger program. (3 marks)

Mark scheme

Award up to 3 marks:

  • Sub programs break a program into smaller sections.
  • They can make code easier to test or maintain.
  • They can reduce repeated code.
  • Accept any other valid point.

3) A points app gives double points during a bonus round. Write a function called doubleNumber that takes a number as a parameter and returns the number multiplied by 2. You may use OCR Exam Reference Language or Python. (4 marks)

Mark scheme

Award up to 4 marks:

  • Defines a function called doubleNumber or a suitable equivalent.
  • Uses a parameter.
  • Multiplies the parameter by 2.
  • Returns the result.
  • Accept any other valid point.

OCR-style pseudocode example:

FUNCTION doubleNumber(number)
    RETURN number * 2
ENDFUNCTION

Python example:

def double_number(number):
    return number * 2

4) A ticket machine has a menu for Buy ticket, Check balance and Exit. Write a procedure called displayMenu that outputs these three menu options. You may use OCR Exam Reference Language or Python. (3 marks)

Mark scheme

Award up to 3 marks:

  • Defines a procedure.
  • Outputs at least three menu options.
  • Does not need to return a value.
  • Accept any other valid point.

OCR-style pseudocode example:

PROCEDURE displayMenu()
    OUTPUT "1. Start"
    OUTPUT "2. Help"
    OUTPUT "3. Quit"
ENDPROCEDURE

Python example:

def display_menu():
    print("1. Start")
    print("2. Help")
    print("3. Quit")

5) Explain the difference between a local variable and a global variable. (4 marks)

Mark scheme

Award up to 4 marks:

  • A local variable is declared or used inside a sub program.
  • A local variable can normally only be used inside that sub program.
  • A global variable is declared outside sub programs.
  • A global variable can be accessed more widely in the program.
  • Accept any other valid point.
