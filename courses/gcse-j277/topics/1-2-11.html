1.2.11 Compression questions

Answer each question, then reveal the hidden mark scheme underneath to check your response.

1) Explain why compression may be needed when storing or transferring files. (3 marks)

Mark scheme

1 mark for each correct point:

  • Compression reduces the size of a file.
  • Smaller files take up less storage space.
  • Smaller files can be transferred more quickly across a network.
  • Smaller files may use less bandwidth.
  • Compression can make it easier to send files as email attachments or upload/download files.
  • Accept any other valid point.

2) Describe the difference between lossy and lossless compression. (4 marks)

Mark scheme

1 mark for each correct point:

  • Lossy compression permanently removes some data from the file.
  • Lossless compression reduces file size without permanently removing data.
  • A lossy compressed file cannot be restored exactly to the original version.
  • A lossless compressed file can be restored exactly to the original version.
  • Lossy compression often creates a smaller file than lossless compression.
  • Accept any other valid point.
