1.6.1.3 Software licences questions

Answer each question, then reveal the hidden mark scheme underneath to check your response.

1) Explain what is meant by a software licence. (3 marks)

Mark scheme

1 mark for each correct point:

  • A software licence is a legal agreement for using software.
  • It sets out what the user is allowed and not allowed to do with the software.
  • It may control copying, installing, modifying or sharing the software.
  • It protects the rights of the software owner or developer.
  • Accept any other valid point.

2) Explain why software licences are needed. (3 marks)

Mark scheme

1 mark for each correct point:

  • They define how software can legally be used.
  • They protect the work and rights of software creators.
  • They help prevent illegal copying or distribution.
  • They can set limits on the number of installations or users.
  • Accept any other valid point.

3) Describe what is meant by proprietary software. (3 marks)

Mark scheme

1 mark for each correct point:

  • Proprietary software is owned by an individual or organisation.
  • The source code is usually not available to users.
  • Users usually need to buy a licence or follow strict licence terms.
  • Users may not be allowed to modify or redistribute the software.
  • Accept any other valid point.

4) Describe what is meant by open source software. (3 marks)

Mark scheme

1 mark for each correct point:

  • Open source software makes its source code available.
  • Users may be allowed to view, modify and share the software.
  • It is released under a licence that sets the conditions of use.
  • It may be free to use, but still has licence terms.
  • Accept any other valid point.

5) Compare proprietary and open source software licences. (4 marks)

Mark scheme

1 mark for each correct point:

  • Proprietary licences usually restrict copying, modification and redistribution.
  • Open source licences usually allow users to view, modify and share source code.
  • Proprietary software source code is usually kept private.
  • Open source software source code is made available.
  • Proprietary software may require payment, while open source software may be free or lower cost.
  • Both still have licence terms that users must follow.
  • Accept any other valid point.

6) A business wants software that it can adapt to meet its own needs. Explain why open source software may be suitable. (4 marks)

Mark scheme

1 mark for each correct point:

  • Open source software gives access to the source code.
  • The business may be allowed to modify the software.
  • This means the software can be adapted to meet the business’s requirements.
  • It may reduce cost compared with some proprietary alternatives.
  • The business must still follow the open source licence terms.
  • Accept any other valid point.

7) A school buys a proprietary software licence for 30 computers. Explain why installing it on 60 computers may be a problem. (4 marks)

Mark scheme

1 mark for each correct point:

  • The licence may only allow installation on 30 computers.
  • Installing it on 60 computers may break the licence agreement.
  • This could be illegal or breach copyright/licence terms.
  • The school may need to buy additional licences.
  • The school could face financial or legal consequences.
  • Accept any other valid point.

9) Explain one advantage and one disadvantage of open source software. (4 marks)

Mark scheme

10) Explain why using software without a valid licence can affect an organisation. (4 marks)

Mark scheme

1 mark for each correct point:

  • The organisation could face fines or legal action.
  • It could damage the organisation’s reputation.
