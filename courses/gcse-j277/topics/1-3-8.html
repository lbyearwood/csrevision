1.3.8 Network layers questions

Answer each question, then reveal the hidden mark scheme underneath to check your response.

1) Explain what is meant by layers in network protocols. (3 marks)

Mark scheme

1 mark for each correct point:

  • Layers split network communication into separate parts.
  • Each layer has a specific role.
  • Layers work together to allow data to be sent and received.
  • A layer can provide services to another layer.
  • Accept any other valid point.

2) Explain why protocols are split into layers. (4 marks)

Mark scheme

1 mark for each correct point:

  • Layers reduce the complexity of network communication.
  • Each layer can focus on a specific task.
  • Layers make protocols easier to develop or maintain.
  • One layer can be changed without changing the whole system.
  • Layers help different hardware and software work together.
  • Layers can help with troubleshooting.
  • Accept any other valid point.

3) Explain one benefit of using layers. (2 marks)

Mark scheme

1 mark for each correct point:

  • They make network communication easier to understand.
  • They make development easier because each layer has a clear role.
  • They support compatibility between systems.
  • They can help identify where faults occur.
  • Accept any other valid point.

4) Explain how layers can help with troubleshooting network problems. (3 marks)

Mark scheme

1 mark for each correct point:

  • Each layer has a specific role.
  • A fault can be narrowed down to one part of the communication process.
  • Different layers can be tested separately.
  • This can make the problem easier to find and fix.
  • Accept any other valid point.

5) A student says, “For OCR GCSE, you must memorise the names and functions of each TCP/IP layer.” Explain why this is incorrect. (2 marks)

Mark scheme

1 mark for each correct point:

  • Students need to understand the concept of layers.
  • Students need to understand the benefits of using layers.
  • The names and functions of each TCP/IP layer are not required.
  • The focus is on how layers are used in protocols.
  • Accept any other valid point.

6) Explain how layers support compatibility between devices and software. (3 marks)

Mark scheme

1 mark for each correct point:

  • Layers provide agreed ways for parts of communication to work.
  • Hardware and software can be designed to follow the same layered approach.
  • Devices from different manufacturers can communicate if they follow the same rules.
  • A layer can be updated if it still works with the other layers.
  • Accept any other valid point.
