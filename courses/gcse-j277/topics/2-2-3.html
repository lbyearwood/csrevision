Paper 2

2.2.3 String manipulation questions

Practise basic string handling, including concatenation, slicing and using strings in simple programs.

1) Describe what is meant by string concatenation. (2 marks)

Mark scheme

Award up to 2 marks:

  • Concatenation joins strings together.
  • It can be used to build a longer string from smaller strings.
  • Accept any other valid point.

2) A school badge system prints a student's full name. Write code that inputs a first name and surname, then outputs them as one full name with a space between them. You may use OCR Exam Reference Language or Python. (4 marks)

Mark scheme

Award up to 4 marks:

  • Inputs the first name.
  • Inputs the surname.
  • Concatenates the names with a space.
  • Outputs the full name.
  • Accept any other valid point.

OCR-style pseudocode example:

firstName = INPUT
surname = INPUT
fullName = firstName + " " + surname
OUTPUT fullName

Python example:

first_name = input()
surname = input()
full_name = first_name + " " + surname
print(full_name)

3) A stock system uses the product code COMPUTER. Write code that stores the string COMPUTER and outputs the first three characters as a short code. You may use OCR Exam Reference Language or Python. (3 marks)

Mark scheme

Award up to 3 marks:

  • Stores the string COMPUTER.
  • Uses slicing or indexing to get the first three characters.
  • Outputs COM.
  • Accept any other valid point.

OCR-style pseudocode example:

word = "COMPUTER"
part = word.substring(0, 3)
OUTPUT part

Python example:

word = "COMPUTER"
part = word[0:3]
print(part)

4) Explain why string slicing may be useful in a program that processes product codes. (3 marks)

Mark scheme

Award up to 3 marks:

  • A product code may contain different pieces of information.
  • Slicing can extract part of the code.
  • The extracted part can be checked or processed separately.
  • Accept any other valid point.

5) A school account system creates usernames automatically. Complete code that creates a username by joining the first three letters of a surname to a year group. You may use OCR Exam Reference Language or Python. (4 marks)

Mark scheme

Award up to 4 marks:

  • Stores or inputs the surname.
  • Stores or inputs the year group.
  • Slices the first three letters of the surname.
  • Concatenates the sliced surname with the year group.
  • Accept any other valid point.

OCR-style pseudocode example:

surname = INPUT
yearGroup = INPUT
username = surname.substring(0, 3) + yearGroup
OUTPUT username

Python example:

surname = input()
year_group = input()
username = surname[0:3] + year_group
print(username)
