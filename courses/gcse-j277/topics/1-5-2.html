1.5.2 Utility software questions

Answer each question, then reveal the hidden mark scheme underneath to check your response.

1) Define what is meant by utility software. (2 marks)

Mark scheme

1 mark for each correct point:

  • Utility software is system software.
  • It helps maintain, manage or optimise a computer system.
  • It performs tasks related to the operation or security of the computer.
  • Accept any other valid point.

2) Identify three examples of utility software. (3 marks)

Mark scheme

1 mark for each correct example:

  • Anti-malware software.
  • Backup software.
  • Encryption software.
  • Disk defragmentation software.
  • Compression software.
  • File management tools.
  • System cleanup tools.
  • Accept any other valid point.

3) Explain the purpose of anti-malware utility software. (3 marks)

Mark scheme

1 mark for each correct point:

  • Anti-malware software scans for malware.
  • It can detect, quarantine or remove malicious software.
  • It helps protect files, programs and the operating system.
  • It needs updates to recognise newer threats.
  • Accept any other valid point.

4) A small business stores customer records on one office computer. Explain the purpose of backup utility software. (3 marks)

Mark scheme

1 mark for each correct point:

  • Backup software creates copies of data.
  • Backups can be used to restore data if files are lost, damaged or deleted.
  • Backups can protect against hardware failure, malware or accidental deletion.
  • Backup software may schedule automatic backups.
  • Accept any other valid point.

5) A student stores coursework on a USB flash drive. Explain the purpose of encryption utility software. (3 marks)

Mark scheme

1 mark for each correct point:

  • Encryption software converts data into an unreadable form.
  • A key or password is needed to decrypt and read the data.
  • It protects data if it is stolen, intercepted or accessed without permission.
  • It is useful for sensitive files and portable devices.
  • Accept any other valid point.

6) A photographer needs to send a large folder of images by email. Explain the purpose of compression utility software. (3 marks)

Mark scheme

1 mark for each correct point:

  • Compression software reduces file size.
  • Smaller files use less storage space.
  • Smaller files can be transferred or uploaded more quickly.
  • It may combine several files into one compressed archive.
  • Accept any other valid point.

7) Explain the purpose of disk defragmentation software. (3 marks)

Mark scheme

1 mark for each correct point:

  • Disk defragmentation reorganises fragmented files on a magnetic hard disk drive.
  • It places parts of files closer together.
  • This can reduce the time taken to read files from a hard disk drive.
  • It is not usually needed for solid state drives and may be unsuitable for them.
  • Accept any other valid point.

8) A student is taking coursework home on a USB flash drive. Explain which utility software could help protect the files. (4 marks)

Mark scheme

1 mark for each correct point:

  • Encryption software could protect the files.
  • It would make the files unreadable without the correct key or password.
  • Backup software could create another copy of the files.
  • This would help restore the work if the USB drive was lost or damaged.
  • Anti-malware software could scan the USB drive for malware.
  • Accept any other valid point.

9) A computer is infected with malware. Explain how utility software could help. (4 marks)

Mark scheme

1 mark for each correct point:

  • Anti-malware software could scan the computer.
  • It could detect the malware.
  • It could quarantine or remove the malware.
  • Backup software may help restore clean copies of files.
  • Keeping anti-malware definitions updated helps detect newer threats.
  • Accept any other valid point.

10) Compare backup software and compression software. (4 marks)

Mark scheme

1 mark for each correct point:

  • Backup software creates copies of files or systems for recovery.
  • Compression software reduces the size of files.
  • Backup software is mainly used to protect against data loss.
  • Compression software is mainly used to save storage space or reduce transfer time.
  • Compressed files may be included in backups to reduce storage requirements.
  • Accept any other valid point.

11) Explain why utility software should be used regularly rather than only after a problem occurs. (4 marks)

Mark scheme

1 mark for each correct point:

  • Regular anti-malware scans can detect threats early.
  • Regular backups reduce the amount of data lost after a problem.
  • Regular updates help security software identify new threats.
  • Regular maintenance can help keep the system running efficiently.
  • Waiting until after a problem may mean data has already been lost or damaged.
  • Accept any other valid point.
