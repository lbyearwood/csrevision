2.2.4 File handling questions

Practise the basic file handling operations expected at GCSE: open, read, write and close.

1) Identify the four basic file handling operations named in the OCR specification. (4 marks)

Mark scheme

Award up to 4 marks:

  • Open.
  • Read.
  • Write.
  • Close.
  • Accept any other valid point.

2) A quiz program writes scores to a text file at the end of a game. Explain why the file should be closed after it has been used by the program. (3 marks)

Mark scheme

Award up to 3 marks:

  • Closing the file can save or finish writing data.
  • It frees the file so it can be used by the operating system or other programs.
  • It reduces the chance of data being lost or corrupted.
  • Accept any other valid point.

3) A club program stores member names in names.txt. Write code that opens the text file, reads one line, outputs it and closes the file. You may use OCR Exam Reference Language or Python. (4 marks)

Mark scheme

Award up to 4 marks:

  • Opens names.txt.
  • Reads a line or item of data from the file.
  • Outputs the data read from the file.
  • Closes the file.
  • Accept any other valid point.

OCR-style pseudocode example:

file = OPEN("names.txt")
name = READ(file)
OUTPUT name
CLOSE(file)

Python example:

file = open("names.txt", "r")
name = file.readline()
print(name)
file.close()

4) A game needs to save a player's score. Write code that opens scores.txt for writing, writes the value 20 and closes the file. You may use OCR Exam Reference Language or Python. (4 marks)

Mark scheme

Award up to 4 marks:

  • Opens scores.txt for writing.
  • Writes 20 to the file.
  • Uses a suitable file variable or handle.
  • Closes the file.
  • Accept any other valid point.

OCR-style pseudocode example:

file = OPENWRITE("scores.txt")
WRITE(file, 20)
CLOSE(file)

Python example:

file = open("scores.txt", "w")
file.write("20")
file.close()

5) Compare reading from a file and writing to a file. (4 marks)

Mark scheme

Award up to 4 marks:

  • Reading gets data from an existing file.
  • Writing stores data in a file.
  • Reading is used when the program needs to use saved data.
  • Writing is used when the program needs to save new or changed data.
  • Accept any other valid point.
