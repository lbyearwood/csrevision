Tasks: 2.4.2 Truth tables

Copy the tables into your book and complete each output carefully.

Student activities

LessonTasksExamQuiz

1) Copy and complete truth tables for A AND B, A OR B and NOT A.

2) Complete a truth table for Q = A AND NOT B. Include a working column for NOT B.

3) Complete a truth table for Q = (A OR B) AND C. Include a working column for A OR B.

4) A system outputs True when a card is valid and the account is not locked. Create a truth table using True and False.

5) Find and correct the error in this row: A = 0, B = 1, A OR B = 0.

6) Explain why a two-input truth table has four rows and a three-input truth table has eight rows.
