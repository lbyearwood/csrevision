Tasks: 2.2.12 Nested Selection constructs

Book-based practice for placing one selection statement inside another to handle several decisions.

Student activities

  1. Write a definition for this topic and underline the three most important words.
  2. Copy the worked example from the lesson and annotate what each line does.
  3. Create a new school, shop or game scenario that uses this topic.
  4. Write pseudocode for your scenario using clear variable names.
  5. Trace your pseudocode with one normal test value and one boundary or unusual value.
