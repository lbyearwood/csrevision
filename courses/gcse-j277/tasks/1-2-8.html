Exam Question - Correct the misconception (3 marks)

A student says, "A computer stores the shape of each letter in a text file." Correct this misconception.

Exam Question answer - Correct the misconception (3 marks)

A computer stores characters using binary codes. [1]

A character set links each character to a unique code. [1]

The visible letter is displayed later by using the stored code. [1]

Exam Question - Compare ASCII and Unicode (4 marks)

Compare ASCII and Unicode. Include one similarity and two differences.

Exam Question answer - Compare ASCII and Unicode (4 marks)

ASCII and Unicode are both character sets that represent characters using codes. [1]

ASCII is smaller and is suitable for basic English text. [1]

Unicode is larger and can represent many languages, symbols and emojis. [1]

Unicode is taught here as using 16 bits per character, so it uses more bits than ASCII. [1]

Exam Question - Decide if enough bits are available (4 marks)

A device needs to represent 100 different symbols. Explain whether 6 bits per character would be enough.

Exam Question answer - Decide if enough bits are available (4 marks)

6 bits would not be enough. [1]

2^6 gives 64 possible codes. [1]

The device needs 100 symbols, so it needs more than 64 codes. [1]

7 bits would be enough because 2^7 gives 128 possible codes. [1]

Exam Question - Explain random ordering problems (3 marks)

Explain one problem that could happen if characters were placed randomly within a character set.

Exam Question answer - Explain random ordering problems (3 marks)

If characters were placed randomly, it would be harder for a computer to compare and sort text. [1]

Sorting names alphabetically would be less straightforward. [1]

This is because the character codes would not follow the same order as the alphabet. [1]
