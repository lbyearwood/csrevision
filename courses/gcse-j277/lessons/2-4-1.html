2.4.1 Logic gate diagrams Lesson

Learn how AND, OR and NOT gates are represented, combined and traced in simple GCSE logic diagrams.

Key ideas

Logic gate diagrams show how binary inputs are processed to produce one output. OCR GCSE students need to recognise AND, OR and NOT gates, follow the flow through a diagram and create diagrams from a short scenario or Boolean expression.

LessonTasksExamQuiz

Gate rules

  • AND outputs 1 only when both inputs are 1.
  • OR outputs 1 when at least one input is 1.
  • NOT outputs the opposite of its single input.

Teacher question: Which gate would you use if an alarm should sound only when the door is open and the system is armed?

Worked example: tracing a diagram

If A = 1, B = 0 and C = 1, first calculate A AND B. The result is 0. Then calculate 0 OR C, which gives Q = 1.

Teacher question: What would Q be if A = 1, B = 1 and C = 0?

Creating diagrams from scenarios

Underline the conditions in the scenario, choose the gate that matches the relationship, and build the diagram from left to right. For example, "the output is 1 if the key is valid and the door is closed" needs an AND gate. If a condition is written as "not", such as "not blocked", use a NOT gate before combining it with another gate.

Teacher question: Sketch the gates for Q = (A OR B) AND NOT C.

Checking and correcting diagrams

When checking a diagram, trace each wire and name each intermediate output. A common mistake is placing NOT in the wrong position. NOT A AND B is different from NOT(A AND B), because the first inverts A before the AND gate and the second inverts the final AND output.
