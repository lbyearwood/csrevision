2.2.15 Exception Handling Lesson

Learn and practise handling errors safely and preventing crashes where appropriate, without making advanced syntax the main GCSE focus.

Core idea

This topic focuses on handling errors safely and preventing crashes where appropriate, without making advanced syntax the main GCSE focus. In exams, marks are usually awarded for clear logic, suitable identifiers and correct use of OCR-style programming ideas.

  • Use precise vocabulary when describing the construct.
  • Trace small examples carefully before writing larger answers.
  • Check that inputs, processing and outputs match the scenario.

Worked example

valid = FALSE
WHILE valid = FALSE
    value = INPUT
    IF value >= 0 THEN
        valid = TRUE
    ENDIF
ENDWHILE

Teacher question: which line does the main processing, and what value would be output for a simple test case?
