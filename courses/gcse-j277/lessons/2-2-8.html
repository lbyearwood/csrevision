2.2.8 Iteration constructs Lesson

Learn and practise repeating instructions using count-controlled and condition-controlled loops.

Core idea

This topic focuses on repeating instructions using count-controlled and condition-controlled loops. In exams, marks are usually awarded for clear logic, suitable identifiers and correct use of OCR-style programming ideas.

  • Use precise vocabulary when describing the construct.
  • Trace small examples carefully before writing larger answers.
  • Check that inputs, processing and outputs match the scenario.

Worked example

FOR count = 1 TO 5
    OUTPUT count
NEXT count

Teacher question: which line does the main processing, and what value would be output for a simple test case?
