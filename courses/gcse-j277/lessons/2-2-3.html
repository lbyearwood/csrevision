2.2.3 Arithmetic operators and Math functions Lesson

Learn and practise using arithmetic operators such as +, -, *, /, DIV, MOD and simple GCSE-level calculations.

Core idea

This topic focuses on using arithmetic operators such as +, -, *, /, DIV, MOD and simple GCSE-level calculations. In exams, marks are usually awarded for clear logic, suitable identifiers and correct use of OCR-style programming ideas.

  • Use precise vocabulary when describing the construct.
  • Trace small examples carefully before writing larger answers.
  • Check that inputs, processing and outputs match the scenario.

Worked example

total = price + delivery
teams = students DIV 4
leftOver = students MOD 4
OUTPUT teams

Teacher question: which line does the main processing, and what value would be output for a simple test case?
