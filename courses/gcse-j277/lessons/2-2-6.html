2.2.6 String functions and string operators Lesson

Learn and practise joining strings, measuring string length, selecting characters, and changing case.

Core idea

This topic focuses on joining strings, measuring string length, selecting characters, and changing case. In exams, marks are usually awarded for clear logic, suitable identifiers and correct use of OCR-style programming ideas.

  • Use precise vocabulary when describing the construct.
  • Trace small examples carefully before writing larger answers.
  • Check that inputs, processing and outputs match the scenario.

Worked example

firstName = INPUT
surname = INPUT
username = LOWER(firstName) + "." + LOWER(surname)
OUTPUT username
