# Specifications

This folder stores course specifications and other official reference documents used to understand content requirements.

## Current Files

- `ocr-gcse-computer-science-j277-specification.pdf` - OCR GCSE Computer Science J277 specification PDF supplied by the user on 2026-06-23.

## Use

- Use specifications to understand required content, vocabulary, assessment scope and exam-style expectations.
- Do not assume the order of specification bullet points is the best lesson order.
- Organise lesson parts in the sequence that best supports student understanding and progression.
- Where no local specification exists, search online for the relevant current specification or ask the user to provide it.
