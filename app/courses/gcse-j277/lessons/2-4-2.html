2.4.2 Truth tables Lesson

Learn how to complete truth tables for AND, OR, NOT and simple combined Boolean expressions.

Key ideas

A truth table lists every possible input combination and the output each combination produces. GCSE questions may use 1 and 0, or True and False, to mean the same Boolean states.

From diagram to table

When a question gives a logic diagram, first write the expression or label the intermediate outputs. Then create one column for each useful intermediate step. This makes it easier to avoid missing a NOT gate or applying a gate in the wrong order.
