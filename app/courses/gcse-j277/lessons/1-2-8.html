Characters

What is a character?

A character is one item of text. It could be a letter, digit, punctuation mark, symbol or space.

The computer does not store the character as a tiny picture. It stores a code that represents the character.

Question

Which of these are characters: A, 7, ?, a space, and a pound sign?

Character sets

Characters need agreed codes

A character set is a collection of characters and the codes used to represent them.

For example, a character set might say that uppercase A uses code 65. The computer can then store that code as binary.

ASCII

ASCII is a smaller character set

ASCII is a character set that represents a limited range of characters. It is useful for basic English letters, digits, punctuation and control characters.

At GCSE, ASCII is often shown using 8-bit binary patterns, such as uppercase A represented as 01000001.

Unicode

Unicode represents many more characters

ASCII

  • Smaller character set.
  • Good for basic English letters, digits and symbols.
  • Limited range compared with modern text needs.

Unicode

  • Much larger character set.
  • Designed for many languages and symbols.
  • May need more bits per character.

Why Unicode matters

Modern computers need to store text from many languages, not just basic English. Unicode gives a much wider range of possible characters.

Each character in a word is represented separately. The character set gives each character a code, and the code is stored as a binary pattern.

Students do not need to memorise ASCII or Unicode codes. The important idea is that text is stored using binary codes.

Misconception check

Correct the common mistakes

Misconception

Characters are stored as tiny pictures.

Correction

Characters are represented using codes from a character set.

Misconception

ASCII and Unicode are the same size.

Correction

Unicode is much larger and can represent many more characters.

Misconception

Unicode means computers no longer use binary.

Correction

Unicode still uses codes that are stored using binary patterns.

Misconception

A character set is only a list of letters.

Correction

A character set includes characters and their codes. Characters can include letters, digits, symbols, punctuation and spaces.

Expected thinking

Exam practice

Exam-style question

Q1 4 marks

Explain how a computer represents characters and why Unicode is useful compared with ASCII.

Answer guidance: Mention that characters are represented by codes from a character set. Explain that the codes are stored as binary. State that ASCII is limited, while Unicode can represent a much wider range of languages and symbols.
