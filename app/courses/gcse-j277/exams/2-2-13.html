2.2.13 Nested Iteration constructs questions

OCR-style practice for using a loop inside another loop, often for rows and columns.

1) Describe the main purpose of Nested Iteration constructs in a program. (3 marks)

Mark scheme

Award up to 3 marks:

  • Gives a clear description linked to the topic.
  • Uses correct programming vocabulary.
  • Applies the idea to a suitable example.
  • Accept any other valid point.

2) A program outputs a 3 by 4 seating grid. Write pseudocode using nested loops. You may use OCR Exam Reference Language or Python. (4 marks)

Mark scheme

Award up to 4 marks:

  • Uses suitable input, output or stored values where relevant.
  • Uses the correct construct or operator for the scenario.
  • Processes data in a logical order.
  • Produces the required result.
  • Accept any other valid point.

OCR-style pseudocode example:

FOR row = 0 TO 2
    FOR column = 0 TO 3
        OUTPUT grid[row][column]
    NEXT column
NEXT row
