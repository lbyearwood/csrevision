2.2.14 2D Lists questions

OCR-style practice for storing data in rows and columns using a list of lists or equivalent table structure.

1) Describe the main purpose of 2D Lists in a program. (3 marks)

Mark scheme

Award up to 3 marks:

  • Gives a clear description linked to the topic.
  • Uses correct programming vocabulary.
  • Applies the idea to a suitable example.
  • Accept any other valid point.

2) A teacher stores marks for 3 students across 2 tests in a 2D list. Explain why a 2D list is suitable and write pseudocode to output one mark. You may use OCR Exam Reference Language or Python. (4 marks)

Mark scheme

Award up to 4 marks:

  • Uses suitable input, output or stored values where relevant.
  • Uses the correct construct or operator for the scenario.
  • Processes data in a logical order.
  • Produces the required result.
  • Accept any other valid point.

OCR-style pseudocode example:

marks = [[5, 7], [8, 6]]
OUTPUT marks[1][0]
