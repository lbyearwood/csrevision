2.2.8 Iteration constructs questions

OCR-style practice for repeating instructions using count-controlled and condition-controlled loops.

1) Describe the main purpose of Iteration constructs in a program. (3 marks)

Mark scheme

Award up to 3 marks:

  • Gives a clear description linked to the topic.
  • Uses correct programming vocabulary.
  • Applies the idea to a suitable example.
  • Accept any other valid point.

2) Write pseudocode that outputs the numbers 1 to 10 using a count-controlled loop. You may use OCR Exam Reference Language or Python. (4 marks)

Mark scheme

Award up to 4 marks:

  • Uses suitable input, output or stored values where relevant.
  • Uses the correct construct or operator for the scenario.
  • Processes data in a logical order.
  • Produces the required result.
  • Accept any other valid point.

OCR-style pseudocode example:

FOR count = 1 TO 5
    OUTPUT count
NEXT count
