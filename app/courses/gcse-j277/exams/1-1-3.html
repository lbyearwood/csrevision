1.1.3 Embedded systems questions

Answer each question, then reveal the hidden mark scheme underneath to check your response.

1) Define what is meant by an embedded system. (2 marks)

Mark scheme

1 mark for each correct point:

  • An embedded system is a computer system built into another device.
  • It is designed to carry out a specific task or small number of specific tasks.
  • It is not usually designed to be a general-purpose computer.
  • It often controls or monitors part of a larger device.
  • Accept any other valid point.

2) A digital thermostat controls the heating in a home. Explain the purpose of the embedded system in this thermostat. (3 marks)

Mark scheme

1 mark for each correct point:

  • Embedded systems allow devices to automatically control specific functions.
  • In the thermostat, the embedded system could monitor temperature and control the heating.
  • They can monitor inputs from sensors.
  • They can process data and produce an output or response.
  • They allow devices to operate without direct user control all the time.
  • They are used to make devices more efficient, reliable or easier to use.
  • Accept any other valid point.

6) A car uses several embedded systems. Explain why embedded systems are suitable for use in a car. (4 marks)

Mark scheme

1 mark for each correct point:

  • Different parts of the car need to carry out specific tasks.
  • Embedded systems can monitor sensors, such as speed, temperature or distance sensors.
  • They can respond quickly to changing conditions.
  • They can control systems automatically, such as braking assistance or engine management.
  • They can be designed to be reliable for repeated use.
  • They can be built into the car without needing a general-purpose computer.
  • Accept any other valid point.

7) Compare an embedded system with a general-purpose computer. (4 marks)

Mark scheme

1 mark for each correct point:

  • An embedded system is designed for a specific task.
  • A general-purpose computer is designed to run many different types of software.
  • An embedded system is usually built into a larger device.
  • A general-purpose computer is usually a separate device, such as a laptop or desktop.
  • An embedded system often has limited hardware resources.
  • A general-purpose computer usually has more processing power, memory and storage.
  • An embedded system may run automatically with little user input.
  • A general-purpose computer usually requires more direct user interaction.
  • Accept any other valid point.

Mark scheme

  • The embedded system is a computer system built into the microwave.
  • It processes inputs, such as buttons or timer settings.
  • It controls outputs, such as the display, turntable motor or heating.
  • A computer system does not need a keyboard, mouse or monitor to process data.
  • Accept any other valid point.

9) Explain why embedded systems are often designed with limited hardware resources. (3 marks)

Mark scheme

1 mark for each correct point:

  • They only need to perform a specific task or small number of tasks.
  • They do not need the same resources as a general-purpose computer.
  • Using less powerful hardware can reduce cost.
  • Using fewer resources can reduce power consumption.
  • Smaller hardware can make it easier to fit inside another device.
  • The system can be designed only for the functions required.
  • Accept any other valid point.

Mark scheme
