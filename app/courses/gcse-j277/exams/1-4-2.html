1.4.2 Identifying and preventing vulnerabilities questions

Answer each question, then reveal the hidden mark scheme underneath to check your response.

1) Explain why organisations should identify vulnerabilities in computer systems. (3 marks)

Mark scheme

1 mark for each correct point:

  • Vulnerabilities can be weaknesses that attackers may exploit.
  • Identifying weaknesses allows the organisation to fix or reduce risks.
  • It can help prevent data loss, malware infection or unauthorised access.
  • It can reduce downtime and improve security.
  • Accept any other valid point.

2) Describe what is meant by penetration testing. (3 marks)

Mark scheme

1 mark for each correct point:

  • Penetration testing is authorised testing of a system’s security.
  • It attempts to find vulnerabilities before attackers do.
  • It may involve trying to gain access to systems or data.
  • The results can be used to improve security.
  • Accept any other valid point.

3) A school laptop may receive infected files from email attachments and USB drives. Explain the purpose of anti-malware software. (3 marks)

Mark scheme

1 mark for each correct point:

  • Anti-malware software detects malware.
  • It can block, quarantine or remove malware.
  • It can scan files, downloads or storage devices.
  • It should be updated so it can recognise newer threats.
  • Accept any other valid point.

4) A small business connects its office network to the Internet. Explain how a firewall helps protect the network. (4 marks)

Mark scheme

1 mark for each correct point:

  • A firewall monitors traffic entering and leaving a network or device.
  • It can block traffic that does not meet security rules.
  • It can help prevent unauthorised access.
  • It can reduce the risk of attacks reaching internal systems.
  • It can be hardware-based, software-based or both.
  • Accept any other valid point.

5) Explain how passwords can help prevent unauthorised access. (3 marks)

Mark scheme

1 mark for each correct point:

  • Passwords are used to authenticate users.
  • Only users who know the correct password should be able to access the account/system.
  • Strong passwords are harder to guess or brute force.
  • Password policies can require length, complexity or regular changes.
  • Accept any other valid point.

6) A company asks staff to use multi-factor authentication. Explain how this improves security. (4 marks)

Mark scheme

1 mark for each correct point:

  • Multi-factor authentication uses more than one method to verify identity.
  • It may require something the user knows, such as a password.
  • It may require something the user has, such as a phone or security token.
  • It makes unauthorised access harder if a password is stolen.
  • Accept any other valid point.

7) Explain why software updates can improve security. (3 marks)

Mark scheme

1 mark for each correct point:

  • Updates can patch known security vulnerabilities.
  • They can fix bugs that attackers might exploit.
  • They may improve protection against new threats.
  • Out-of-date software may be easier to attack.
  • Accept any other valid point.

8) In a school system, teachers can edit marks but students can only view their own results. Explain how access levels can help protect data. (4 marks)

Mark scheme

1 mark for each correct point:

  • Access levels control what users are allowed to see or do.
  • Users should only have access to the data or systems they need.
  • This reduces the risk of accidental or deliberate misuse.
  • It can stop lower-level users from changing important settings or viewing sensitive data.
  • Accept any other valid point.

9) Explain why encryption is used to protect data. (3 marks)

Mark scheme

1 mark for each correct point:

  • Encryption converts data into an unreadable form without the correct key.
  • It protects data if it is intercepted or stolen.
  • Only authorised users with the key should be able to read the original data.
  • It is useful for sensitive data stored or transmitted across networks.
  • Accept any other valid point.

10) A laptop containing personal data is lost on a train. Explain how encryption could reduce the risk to the organisation. (4 marks)

Mark scheme

1 mark for each correct point:

  • Encryption would make the stored data unreadable without the key/password.
  • A person who finds or steals the laptop should not be able to easily read the data.
  • This reduces the risk of personal data being exposed.
  • It can help protect the organisation from data breaches and legal consequences.
  • Accept any other valid point.

11) Compare anti-malware software and a firewall. (4 marks)

Mark scheme

1 mark for each correct point:

  • Anti-malware detects, blocks or removes malicious software.
  • A firewall monitors and filters network traffic.
  • Anti-malware focuses on threats such as viruses, worms and trojans.
  • A firewall can block unauthorised traffic or connections.
  • Both help protect computer systems and networks.
  • Accept any other valid point.

12) A school wants to reduce the risk of unauthorised access to its network. Explain four security measures it could use. (6 marks)

Mark scheme
