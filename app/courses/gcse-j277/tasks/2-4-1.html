Tasks: 2.4.1 Logic gate diagrams

Copy the activities into your book and answer using AND, OR and NOT logic gate diagrams.

Student activities

LessonTasksExamQuiz

1) Draw and label the symbols for AND, OR and NOT gates. Next to each symbol, write the rule for when it outputs 1.

2) Draw a logic diagram for Q = A AND B. Then draw a second diagram for Q = A OR B. Explain the difference in one sentence.

3) Draw a logic diagram for Q = (A AND B) OR C. Label the intermediate output from the AND gate.

4) Draw a logic diagram for Q = A AND NOT B. Explain where the NOT gate must be placed.

5) A security light turns on when the motion sensor is active OR the manual switch is on, but only when it is dark. Draw a logic diagram for this scenario.

6) Correct this faulty expression-diagram description: "Q = NOT A AND B" has a NOT gate after the AND gate. Explain the correction.
