1.3.2.3 Layers

Answer each question, then reveal the hidden mark scheme underneath to check your response.

1) Explain what is meant by layers in network communication. (3 marks)

Mark scheme

1 mark for each correct point:

  • Layers split network communication into separate stages.
  • Each layer has a specific role or set of responsibilities.
  • Layers work together to allow data to be sent across a network.
  • A layer provides services to the layer above or below it.
  • Accept any other valid point.

2) Explain why network communication is divided into layers. (4 marks)

Mark scheme

1 mark for each correct point:

  • It breaks a complex process into smaller parts.
  • Each layer can be developed or changed separately.
  • It helps different hardware and software work together using agreed standards.
  • It makes troubleshooting easier because problems can be located within a layer.
  • It allows protocols at different layers to work together.
  • Accept any other valid point.

3) Describe the role of the application layer. (3 marks)

Mark scheme

1 mark for each correct point:

  • The application layer provides network services to user applications.
  • It includes protocols used by applications, such as HTTP, HTTPS, FTP, SMTP, POP or IMAP.
  • It is involved when users access websites, email or file transfer services.
  • Accept any other valid point.

4) Describe the role of the transport layer. (3 marks)

Mark scheme

1 mark for each correct point:

  • The transport layer manages end-to-end communication between devices.
  • It can split data into packets or segments.
  • It can help ensure data arrives in the correct order and can be reassembled.
  • TCP operates at the transport layer.
  • Accept any other valid point.

5) Describe the role of the Internet layer. (3 marks)

Mark scheme

1 mark for each correct point:

  • The Internet layer is responsible for addressing and routing packets.
  • It uses IP addresses to help send data to the correct destination.
  • It decides the route packets take across networks.
  • IP operates at the Internet layer.
  • Accept any other valid point.

6) Describe the role of the link layer. (3 marks)

Mark scheme

1 mark for each correct point:

  • The link layer is responsible for communication across the physical/local network link.
  • It deals with how data is transmitted over the network medium.
  • It can involve MAC addresses and network hardware.
  • It is used to move data between devices on the same local network.
  • Accept any other valid point.

7) Explain how data moves through layers when a user sends data across a network. (5 marks)

Mark scheme

1 mark for each correct point:

  • Data starts at the application layer when an application creates or requests data.
  • The transport layer prepares the data for transmission, such as splitting it into packets/segments.
  • The Internet layer adds addressing/routing information such as IP addresses.
  • The link layer prepares the data for transmission over the local network medium.
  • At the receiving device, the process is reversed and data is passed back up the layers.
  • Accept any other valid point.

8) A user enters a web address into a browser. Explain how protocols and layers may be involved in loading the web page. (6 marks)

Indicative content:

  • The browser uses an application layer protocol such as HTTP or HTTPS.

9) Explain one benefit of using layered models for network troubleshooting. (3 marks)

Mark scheme

1 mark for each correct point:

  • A layered model separates network communication into stages.
  • This helps technicians narrow down where a fault may be occurring.
  • For example, a cable fault may be linked to a lower layer, while a website problem may be linked to an application layer service.
  • It makes testing and fixing network issues more organised.
  • Accept any other valid point.
