2.1.5 Pseudocode

Practise reading and writing language-independent algorithm steps.

1) Define what is meant by pseudocode. (2 marks)

Mark scheme

1 mark for each correct point:

  • Pseudocode is a way of writing an algorithm using structured English-like statements.
  • It is not tied to one specific programming language.
  • It can be used to plan a program before coding.
  • Accept any other valid point.

2) A teacher wants students to plan a quiz algorithm before choosing a programming language. Explain one advantage of using pseudocode for this design. (3 marks)

Mark scheme

1 mark for each correct point:

  • It focuses on the logic of the solution rather than exact syntax.
  • It can be understood by programmers using different languages.
  • It helps plan the algorithm before implementation.
  • It can make errors in the logic easier to spot.
  • Accept any other valid point.

3) A program repeatedly asks for a password until the correct password is entered. Write pseudocode for this algorithm. (5 marks)

Mark scheme

Award up to 5 marks:

  • Initialises or stores the correct password.
  • Inputs a password from the user.
  • Uses iteration to repeat while the password is incorrect.
  • Inputs another password inside the loop.
  • Outputs a suitable success message when the correct password is entered.
  • Accept any other valid pseudocode.

4) A password-check algorithm uses an IF statement and a WHILE loop. Identify these two common pseudocode constructs and describe what each one does. (4 marks)

Mark scheme

Award up to 4 marks:

  • Selection, such as IF, chooses between different paths.
  • Iteration, such as WHILE or FOR, repeats instructions.
  • Input gets data from the user or another source.
  • Output displays or returns information.
  • Assignment stores a value in a variable.
  • Accept any other valid point.
