Paper 2

2.1.3 Algorithmic thinking

Practise describing clear, ordered steps that solve a problem.

1) Define what is meant by an algorithm. (2 marks)

Mark scheme

1 mark for each correct point:

  • An algorithm is a set of instructions or steps.
  • The steps are followed to solve a problem or complete a task.
  • An algorithm should be precise and ordered.
  • Accept any other valid point.

2) Explain why the order of steps in an algorithm is important. (3 marks)

Mark scheme

1 mark for each correct point:

  • The computer follows instructions in the order given.
  • Changing the order can change the result of the algorithm.
  • Some steps depend on earlier steps having already happened.
  • Incorrect order can cause errors or incorrect output.
  • Accept any other valid point.

3) A program asks for two numbers and displays the larger number. Describe an algorithm for this task. (4 marks)

Mark scheme

Award up to 4 marks:

  • Input the first number.
  • Input the second number.
  • Compare the two numbers.
  • If the first number is larger, output the first number.
  • If the second number is larger, output the second number.
  • A valid answer may also handle equal numbers.
  • Accept any other valid algorithm.

4) Explain why an algorithm should be tested before it is turned into a final program. (3 marks)

Mark scheme

1 mark for each correct point:

  • Testing can identify logic errors in the algorithm.
  • It helps check that the algorithm produces the expected output.
  • It is easier to fix problems before or during implementation.
  • Testing can check different inputs and edge cases.
  • Accept any other valid point.
