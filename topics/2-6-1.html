2.6.1 Writing basic programs questions

Practise writing short code fragments using input, output, variables, operators, selection, loops and simple data access.

1) A school welcome screen asks a student to enter their name. Write code that inputs the name and outputs Hello followed by the name. You may use OCR Exam Reference Language or Python. (2 marks)

Mark scheme

Award up to 2 marks:

  • Inputs or stores the user's name.
  • Outputs a greeting that includes the stored name.
  • Accept OCR Exam Reference Language or a high-level programming language.
  • Accept any other valid point.

OCR-style pseudocode example:

name = input("Enter name")
print("Hello " + name)

Python example:

name = input("Enter name")
print("Hello " + name)

2) A garden design program calculates the area of a rectangular lawn. Complete code to calculate the area using length and width variables, then output the area. You may use OCR Exam Reference Language or Python. (3 marks)

Mark scheme

Award up to 3 marks:

  • Uses suitable variables for length and width.
  • Calculates area using length multiplied by width.
  • Outputs the calculated area.
  • Accept OCR Exam Reference Language or a high-level programming language.
  • Accept any other valid point.

OCR-style pseudocode example:

area = length * width
print(area)

Python example:

area = length * width
print(area)

3) A quiz program inputs a student's mark. Write code that inputs the mark and outputs Pass if the mark is 50 or more. You may use OCR Exam Reference Language or Python. (3 marks)

Mark scheme

Award up to 3 marks:

  • Inputs or stores the mark.
  • Uses a comparison to test whether the mark is greater than or equal to 50.
  • Outputs Pass when the condition is true.
  • Accept OCR Exam Reference Language or a high-level programming language.
  • Accept any other valid point.

OCR-style pseudocode example:

mark = input("Enter mark")
if mark >= 50 then
    print("Pass")
endif

Python example:

mark = int(input("Enter mark"))
if mark >= 50:
    print("Pass")

4) A fitness app displays days 1 to 5 of a challenge. Write code that uses a loop to output the numbers 1 to 5. You may use OCR Exam Reference Language or Python. (3 marks)

Mark scheme

Award up to 3 marks:

  • Uses a suitable loop.
  • Starts at or outputs the value 1.
  • Repeats until the value 5 has been output.
  • Accept OCR Exam Reference Language or a high-level programming language.
  • Accept any other valid point.

OCR-style pseudocode example:

for day = 1 to 5
    print(day)
next day

Python example:

for day in range(1, 6):
    print(day)

5) A club program stores the names Ali, Beth and Chen. Write code that stores the three names in a list or array and outputs the second name. You may use OCR Exam Reference Language or Python. (3 marks)

Mark scheme

Award up to 3 marks:

  • Creates or uses a suitable list or array.
  • Stores three names in the list or array.
  • Outputs the second item using a valid index for the chosen language.
  • Accept zero-based or one-based indexing if it matches the language used.
  • Accept any other valid point.

OCR-style pseudocode example:

names = ["Ali", "Beth", "Chen"]
print(names[1])

Python example:

names = ["Ali", "Beth", "Chen"]
print(names[1])

6) A points app receives a whole number from a text box. Write code that inputs the number as text, casts it to an integer, adds 10 bonus points and outputs the result. You may use OCR Exam Reference Language or Python. (4 marks)

Mark scheme

Award up to 4 marks:

  • Inputs or stores the value as text.
  • Casts or converts the value to an integer.
  • Adds 10 to the integer value.
  • Outputs the result.
  • Accept OCR Exam Reference Language or a high-level programming language.
  • Accept any other valid point.

OCR-style pseudocode example:

pointsText = input("Enter points")
points = int(pointsText)
total = points + 10
print(total)

Python example:

points_text = input("Enter points")
points = int(points_text)
total = points + 10
print(total)
