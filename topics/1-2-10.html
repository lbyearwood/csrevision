1.2.10 Sound questions

Answer each question, then reveal the hidden mark scheme underneath to check your response.

1) Explain why analogue sound must be converted into binary before it can be stored by a computer. (3 marks)

Mark scheme

1 mark for each correct point:

  • Sound in the real world is analogue.
  • Computers process and store data in binary.
  • The sound must be sampled so it can be represented as digital data.
  • Each sample is stored as a binary value.
  • Converting sound to binary allows it to be saved, processed and played back by a computer.
  • Accept any other valid point.

2) Describe how sound is sampled and stored in digital form. (4 marks)

Mark scheme

1 mark for each correct point:

  • The sound wave is measured at regular intervals.
  • Each measurement is called a sample.
  • Each sample records the amplitude/height of the sound wave at that point.
  • Each sample is converted into a binary value.
  • The collection of binary values represents the sound digitally.
  • The sound can then be stored as a sound file.
  • Accept any other valid point.

3) Explain what is meant by sample rate. (3 marks)

Mark scheme

1 mark for each correct point:

  • Sample rate is how many samples are taken each second.
  • Sample rate is measured in Hertz/Hz.
  • A higher sample rate means more measurements of the sound wave are taken each second.
  • A higher sample rate can improve the accuracy/quality of the stored sound.
  • A higher sample rate usually increases the file size.
  • Accept any other valid point.

Mark scheme

  • A higher bit depth allows more possible values for each sample.
  • A higher bit depth can represent the sound more accurately.
  • A higher bit depth can improve playback quality.
  • A higher bit depth usually increases the file size.
  • Accept any other valid point.

Mark scheme

9) Calculate the file size in bits for a sound file with a sample rate of 8000 Hz, a duration of 10 seconds and a bit depth of 8 bits. Show your working. (3 marks)

Mark scheme

1 mark for each correct point:

  • Uses correct formula: sample rate x duration x bit depth.
  • Substitution: 8000 x 10 x 8.
  • Correct answer: 640000 bits.
  • Accept any other valid point.
