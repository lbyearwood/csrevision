1.3.1.1 Network hardware

Answer each question, then reveal the hidden mark scheme underneath to check your response.

1) Identify two pieces of hardware used in a network. (2 marks)

Mark scheme

1 mark for each correct item:

  • Router.
  • Switch.
  • Wireless access point.
  • Network Interface Controller/Card / NIC.
  • Transmission media, such as Ethernet cables or fibre optic cables.
  • Accept any other valid point.

2) A small business network needs to connect its office LAN to the Internet. Explain the purpose of a router in this network. (3 marks)

Mark scheme

1 mark for each correct point:

  • A router connects networks together.
  • It forwards data packets between networks.
  • It can connect a LAN to the Internet.
  • It uses IP addresses to help direct data to the correct destination.
  • Accept any other valid point.

3) A school computer room has 30 desktop computers connected by cables. Explain the purpose of a switch in this network. (3 marks)

Mark scheme

1 mark for each correct point:

  • A switch connects devices together on a LAN.
  • It forwards data to the correct device on the network.
  • It can use MAC addresses to send data to the correct destination.
  • It reduces unnecessary network traffic compared with sending data to every device.
  • Accept any other valid point.

4) A library wants visitors to connect phones and tablets to the network using Wi-Fi. Explain the purpose of a wireless access point. (3 marks)

Mark scheme

1 mark for each correct point:

  • A wireless access point allows wireless devices to connect to a network.
  • It sends and receives wireless signals.
  • It can connect wireless devices to a wired network.
  • It extends or provides wireless network coverage.
  • Accept any other valid point.

5) A desktop computer is being added to a wired network. Explain the purpose of a Network Interface Controller/Card in the computer. (3 marks)

Mark scheme

1 mark for each correct point:

  • A NIC allows a device to connect to a network.
  • It may allow wired or wireless network connection.
  • It has a MAC address used to identify the device on a network.
  • It handles communication between the device and the network medium.
  • Accept any other valid point.

6) Compare the role of a router and a switch. (4 marks)

Mark scheme

1 mark for each correct point:

  • A router connects different networks together.
  • A switch connects devices together within a LAN.
  • A router can forward data between a LAN and the Internet.
  • A switch forwards data to devices on the local network.
  • Routers often use IP addresses to route data between networks.
  • Switches often use MAC addresses to forward data within a LAN.
  • Accept any other valid point.

7) A home user wants several devices to connect to the Internet using Wi-Fi. Identify the network hardware needed and explain its role. (4 marks)

Mark scheme

1 mark for each correct point:

  • A router would be needed to connect the home network to the Internet.
  • A wireless access point would allow wireless devices to connect.
  • A home router may include a built-in wireless access point.
  • NICs are needed in devices so they can connect to the network.
  • The answer should link the hardware to wireless Internet access.
  • Accept any other valid point.

8) A school adds a new computer room with 30 desktop computers. Explain why a switch would be useful. (4 marks)

Mark scheme

1 mark for each correct point:

  • A switch can connect the 30 computers together on the LAN.
  • It can connect the computers to shared resources such as printers or servers.
  • It forwards data to the correct device.
  • It helps reduce unnecessary traffic compared with broadcasting to all devices.
  • It allows the school network to be expanded.
  • Accept any other valid point.

9) Explain why transmission media is needed in a network. (3 marks)

Mark scheme

1 mark for each correct point:

  • Transmission media carries data between devices.
  • It may be wired, such as Ethernet or fibre optic cable.
  • It may be wireless, such as radio waves/Wi-Fi.
  • The type of media can affect speed, reliability and range.
  • Accept any other valid point.

10) Explain how network hardware can affect network performance. (5 marks)

Mark scheme

1 mark for each correct point:

  • Outdated or low-quality hardware can limit speed.
  • A switch or router with low bandwidth may create a bottleneck.
  • Wireless access points can be affected by range or interference.
  • Transmission media can affect data transfer speed.
  • Faster hardware can support more data being transferred.
  • Network performance also depends on the number of users and traffic.
  • Accept any other valid point.
