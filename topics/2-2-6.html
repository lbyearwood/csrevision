1) Write an SQL statement to display the FirstName and TutorGroup fields for every record in the Students table. (3 marks)

Mark scheme

Award up to 3 marks:

  • Uses SELECT with FirstName and TutorGroup.
  • Uses FROM Students.
  • Correct statement structure.
  • Accept any other valid SQL statement.
SELECT FirstName, TutorGroup
FROM Students

2) Write an SQL statement to display all fields for every record in the Books table. (2 marks)

Mark scheme

Award up to 2 marks:

  • Uses SELECT * to select every field.
  • Uses FROM Books.
  • Accept any other valid SQL statement.
SELECT *
FROM Books

3) Write an SQL statement to display the FirstName and Points fields for students with more than 40 points. (4 marks)

Mark scheme

Award up to 4 marks:

  • Uses SELECT FirstName, Points.
  • Uses FROM Students.
  • Uses WHERE.
  • Uses the correct comparison: Points > 40.
  • Accept any other valid SQL statement.
SELECT FirstName, Points
FROM Students
WHERE Points > 40

4) Write an SQL statement to display all fields for students in YearGroup 10. (3 marks)

Mark scheme

Award up to 3 marks:

  • Uses SELECT *.
  • Uses FROM Students.
  • Uses WHERE YearGroup = 10.
  • Accept any other valid SQL statement.
SELECT *
FROM Students
WHERE YearGroup = 10

5) Write an SQL statement to display the Title and Author fields for books in the Computing genre. (4 marks)

Mark scheme

Award up to 4 marks:

  • Uses SELECT Title, Author.
  • Uses FROM Books.
  • Uses WHERE.
  • Correctly tests Genre = "Computing".
  • Accept single quotes around text values.
  • Accept any other valid SQL statement.
SELECT Title, Author
FROM Books
WHERE Genre = "Computing"

6) Write an SQL statement to display the Title and Copies fields for books with 2 or fewer copies. (4 marks)

Mark scheme

Award up to 4 marks:

  • Uses SELECT Title, Copies.
  • Uses FROM Books.
  • Uses WHERE.
  • Uses the correct comparison: Copies <= 2.
  • Accept any other valid SQL statement.
SELECT Title, Copies
FROM Books
WHERE Copies <= 2

7) Write an SQL statement to display FirstName, House and Points for students in Red house with at least 30 points. (5 marks)

Mark scheme

Award up to 5 marks:

  • Uses SELECT FirstName, House, Points.
  • Uses FROM Students.
  • Uses WHERE.
  • Correctly tests House = "Red".
  • Uses AND with Points >= 30.
  • Accept single quotes around text values.
  • Accept any other valid SQL statement.
SELECT FirstName, House, Points
FROM Students
WHERE House = "Red" AND Points >= 30

8) Write an SQL statement to display all fields for students who are in Blue house or Yellow house. (4 marks)

Mark scheme

Award up to 4 marks:

  • Uses SELECT *.
  • Uses FROM Students.
  • Uses WHERE.
  • Uses OR to test House = "Blue" or House = "Yellow".
  • Accept single quotes around text values.
  • Accept any other valid SQL statement.
SELECT *
FROM Students
WHERE House = "Blue" OR House = "Yellow"

9) Write an SQL statement to display FirstName and TutorGroup for students who are not in YearGroup 11. (4 marks)

Mark scheme

Award up to 4 marks:

  • Uses SELECT FirstName, TutorGroup.
  • Uses FROM Students.
  • Uses WHERE.
  • Correctly excludes YearGroup 11 using NOT or !=.
  • Accept any other valid SQL statement.
SELECT FirstName, TutorGroup
FROM Students
WHERE NOT YearGroup = 11

Also accept:

SELECT FirstName, TutorGroup
FROM Students
WHERE YearGroup != 11

10) Write an SQL statement to display the Title and Author fields for books where the Title starts with Python. (4 marks)

Mark scheme

Award up to 4 marks:

  • Uses SELECT Title, Author.
  • Uses FROM Books.
  • Uses WHERE with LIKE.
  • Uses a valid wildcard pattern such as "Python%".
  • Accept single quotes around text values.
  • Accept any other valid SQL statement.
SELECT Title, Author
FROM Books
WHERE Title LIKE "Python%"

11) Write an SQL statement to display Title, YearPublished and Copies for available books published after 2020, sorted by YearPublished. (5 marks)

Mark scheme

Award up to 5 marks:

  • Uses SELECT Title, YearPublished, Copies.
  • Uses FROM Books.
  • Correctly tests Available = "Yes".
  • Uses AND with YearPublished > 2020.
  • Uses SORTBY YearPublished.
  • Accept single quotes around text values.
  • Accept any other valid SQL statement.
SELECT Title, YearPublished, Copies
FROM Books
WHERE Available = "Yes" AND YearPublished > 2020
SORTBY YearPublished
