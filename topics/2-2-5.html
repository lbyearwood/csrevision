Paper 2

2.2.5 Records questions

Practise using records to store related fields about one item, person or object.

1) Define what is meant by a record. (2 marks)

Mark scheme

Award up to 2 marks:

  • A record stores related data about one item or entity.
  • A record is made up of fields.
  • Accept any other valid point.

2) A school information system stores one record for each student. Identify four fields that could be stored in a Student record. (4 marks)

Mark scheme

Award up to 4 marks:

  • StudentID.
  • FirstName.
  • Surname.
  • TutorGroup.
  • DateOfBirth.
  • House.
  • Accept any other valid field.

3) Explain why records are useful when storing data about library books. (3 marks)

Mark scheme

Award up to 3 marks:

  • A record can store all data about one book together.
  • Fields can store details such as title, author and ISBN.
  • This makes the data easier to organise, search or update.
  • Accept any other valid point.

4) A library program needs to store details about one book. Write code that creates or assigns values to a Book record with Title, Author and Copies fields. You may use OCR Exam Reference Language or Python. (4 marks)

Mark scheme

Award up to 4 marks:

  • Creates or uses a record-like structure.
  • Stores a title field.
  • Stores an author field.
  • Stores a copies field.
  • Accept any other valid point.

OCR-style pseudocode example:

Book.Title = "Python Basics"
Book.Author = "Lina Patel"
Book.Copies = 7
OUTPUT Book.Title

Python example:

book = {"Title": "Python Basics", "Author": "Lina Patel", "Copies": 7}
print(book["Title"])

5) A library receives one extra copy of a book. Complete code that updates the Copies field of a Book record by adding 1. You may use OCR Exam Reference Language or Python. (3 marks)

Mark scheme

Award up to 3 marks:

  • Accesses the Copies field.
  • Adds 1 to the existing value.
  • Stores the updated value back in the record.
  • Accept any other valid point.

OCR-style pseudocode example:

Book.Copies = Book.Copies + 1

Python example:

book["Copies"] = book["Copies"] + 1
