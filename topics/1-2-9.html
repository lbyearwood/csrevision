1.2.9 Images questions

Answer each question, then reveal the hidden mark scheme underneath to check your response.

1) Describe how bitmap images are represented by a computer. (3 marks)

Mark scheme

1 mark for each correct point:

  • A bitmap image is made up of pixels.
  • Each pixel has a colour.
  • Each colour is represented using a binary code.
  • The binary data for all pixels is stored so the image can be recreated.
  • The computer processes and stores the image data in binary.
  • Accept any other valid point.

2) Explain the relationship between pixels and image resolution. (3 marks)

Mark scheme

1 mark for each correct point:

  • Resolution is the number of pixels in an image.
  • Resolution is commonly shown as width by height in pixels.
  • A higher resolution means more pixels are used.
  • More pixels can allow more detail to be shown.
  • A higher resolution usually increases the file size.
  • Accept any other valid point.

3) Explain how colour depth affects an image. (4 marks)

Mark scheme

1 mark for each correct point:

  • Colour depth is the number of bits used to represent the colour of each pixel.
  • A higher colour depth allows more colours to be represented.
  • More colours can improve image quality or make the image look more realistic.
  • A higher colour depth increases the number of bits needed per pixel.
  • This increases the file size of the image.
  • Accept any other valid point.

4) Explain how resolution affects image quality and file size. (4 marks)

Mark scheme

1 mark for each correct point:

  • A higher resolution uses more pixels.
  • More pixels can make the image show more detail.
  • This can improve image quality.
  • More pixels require more data to be stored.
  • This increases the file size.
  • A lower resolution usually reduces detail and file size.
  • Accept any other valid point.

5) Compare the effect of increasing colour depth and increasing resolution. (4 marks)

Mark scheme

1 mark for each correct point:

  • Increasing colour depth increases the number of colours that can be represented.
  • Increasing resolution increases the number of pixels in the image.
  • Both can improve image quality.
  • Both can increase file size.
  • Colour depth affects the number of bits per pixel.
  • Resolution affects the number of pixels that must be stored.
  • Accept any other valid point.

6) Explain why metadata is stored with an image file. (3 marks)

Mark scheme

1 mark for each correct point:

  • Metadata stores additional information about the image.
  • Metadata may include image height and width.
  • Metadata may include colour depth, file type, date created or camera settings.
  • Metadata helps software correctly display or manage the image.
  • Metadata is not the main pixel data of the image.
  • Accept any other valid point.

Mark scheme

1 mark for each correct point:

  • Colour depth is about the number of bits used for each pixel.
  • Colour depth affects how many colours can be represented.

  • An image can have a higher colour depth without having more pixels.
  • The student has confused colour depth with resolution.

8) A student says, “Increasing the resolution will usually reduce the image file size.” Explain why this statement is incorrect. (3 marks)

Mark scheme

1 mark for each correct point:

  • Increasing resolution means increasing the number of pixels.
  • More pixels need more binary data to be stored.
  • This usually increases the file size.
  • Reducing resolution would usually reduce the file size.
  • The file size may also depend on colour depth and compression.
  • Accept any other valid point.

9) Calculate the file size in bits for an image that is 100 pixels wide, 80 pixels high and has a colour depth of 8 bits. (3 marks)

Mark scheme

Award up to 3 marks:

  • Uses correct formula: width x height x colour depth.
  • Calculates number of pixels: 100 x 80 = 8000.
  • Calculates file size: 8000 x 8 = 64000 bits.
  • Accept equivalent working.

10) Calculate the file size in bytes for an image that is 50 pixels wide, 40 pixels high and has a colour depth of 16 bits. (4 marks)

Mark scheme

Award up to 4 marks:

  • Uses correct formula: width x height x colour depth.
  • Calculates number of pixels: 50 x 40 = 2000.
  • Calculates file size in bits: 2000 x 16 = 32000 bits.
  • Converts to bytes: 32000 / 8 = 4000 bytes.
  • Accept equivalent working.

11) Explain why an image with a higher resolution and higher colour depth would need more storage space. (5 marks)

Mark scheme

1 mark for each correct point:

  • Higher resolution means more pixels are stored.
  • Higher colour depth means more bits are stored for each pixel.
  • More pixels increase the amount of binary data.
  • More bits per pixel increase the amount of binary data.
  • The total file size depends on width x height x colour depth.
  • Therefore, increasing both resolution and colour depth increases the storage needed.
  • Accept any other valid point.
