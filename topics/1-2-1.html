1.2.1 Primary storage

Answer each question, then reveal the hidden mark scheme underneath to check your response.

1) A student opens a web browser and a document editor on a laptop. Explain why the laptop needs primary storage while these programs are running. (3 marks)

Mark scheme

1 mark for each correct point:

  • Primary storage holds data and instructions currently being used by the CPU.
  • The browser and document editor instructions/data can be held in primary storage while in use.
  • The CPU needs quick access to data and instructions while programs are running.
  • Primary storage is faster to access than secondary storage.
  • Without primary storage, the CPU would not be able to directly access the data and instructions needed for processing.
  • Accept any other valid point.

2) Compare RAM and ROM. (4 marks)

Mark scheme

1 mark for each correct point:

  • RAM is volatile, whereas ROM is non-volatile.
  • RAM stores data, instructions and programs currently in use.
  • ROM stores permanent instructions needed to start the computer.
  • The contents of RAM are lost when the computer is turned off.
  • The contents of ROM remain when the computer is turned off.
  • RAM can usually be written to and changed, whereas ROM is usually read-only.
  • Accept any other valid point.

3) A school computer has several programs open during a lesson. Explain the purpose of RAM in this computer system. (3 marks)

Mark scheme

1 mark for each correct point:

  • RAM stores data and instructions currently being used by the CPU.
  • RAM stores programs that are currently running.
  • The open lesson programs are loaded into RAM so the CPU can access them quickly.
  • RAM allows the CPU to access data and instructions quickly.
  • RAM is volatile, so its contents are lost when power is turned off.
  • More RAM can allow more programs or larger files to be used at the same time.
  • Accept any other valid point.

4) When a laptop is switched on, it starts the boot process before the operating system loads. Explain the purpose of ROM in this process. (3 marks)

Mark scheme

1 mark for each correct point:

  • ROM stores permanent instructions needed by the computer.
  • ROM stores boot-up instructions or firmware.
  • These start-up instructions can be used before the operating system has loaded.
  • ROM is used when the computer starts up.
  • ROM is non-volatile, so its contents remain when power is turned off.
  • ROM cannot usually be changed by the user.
  • Accept any other valid point.

5) A student says, “RAM and ROM are both types of memory, so they do the same job.” Explain why this statement is incorrect. (4 marks)

Mark scheme

1 mark for each correct point:

  • RAM and ROM have different purposes.
  • RAM stores data and instructions currently in use.
  • ROM stores permanent instructions such as boot-up instructions.
  • RAM is volatile, so its contents are lost when power is turned off.
  • ROM is non-volatile, so its contents remain when power is turned off.
  • RAM can usually be changed, whereas ROM is usually read-only.
  • Accept any other valid point.

6) Explain why virtual memory may be needed in a computer system. (3 marks)

Mark scheme

1 mark for each correct point:

  • Virtual memory may be needed when RAM is full.
  • The computer uses part of secondary storage as if it were RAM.
  • Data not currently needed can be moved from RAM to secondary storage.
  • This allows the computer to continue running when there is not enough RAM available.
  • It can allow more programs to be open than would fit in RAM alone.
  • Accept any other valid point.

7) Describe how virtual memory works when RAM is full. (4 marks)

Mark scheme

1 mark for each correct point:

  • When RAM is full, the operating system uses part of secondary storage as virtual memory.
  • Data or instructions not currently needed are moved out of RAM.
  • This data or these instructions are stored temporarily on secondary storage.
  • When needed again, the data or instructions are moved back into RAM.
  • This transfer between RAM and secondary storage allows the system to keep running.
  • Accept any other valid point.

8) Explain why using virtual memory can reduce system performance. (3 marks)

Mark scheme

1 mark for each correct point:

  • Virtual memory uses secondary storage.
  • Secondary storage is slower to access than RAM.
  • Moving data between RAM and secondary storage takes time.
  • The CPU may have to wait longer for data and instructions.
  • Excessive use of virtual memory can make the system run slowly.
  • Accept any other valid point.

9) Explain the purpose of cache in a computer system. (3 marks)

Mark scheme

1 mark for each correct point:

  • Cache stores frequently used data and instructions.
  • Cache is very fast memory located close to or inside the CPU.
  • Cache allows the CPU to access commonly needed data and instructions more quickly.
  • Cache reduces the need for the CPU to access slower RAM.
  • This can improve processing speed.
  • Accept any other valid point.

11) A computer starts running slowly when several large programs are open at the same time. Explain how RAM and virtual memory are involved in this situation. (5 marks)

Mark scheme

1 mark for each correct point:

  • Running several large programs requires a large amount of RAM.
  • RAM stores the programs, data and instructions currently in use.
  • If RAM becomes full, the system may use virtual memory.
  • Virtual memory moves data between RAM and secondary storage.
  • Secondary storage is slower than RAM.
  • This can cause the computer to run more slowly.
  • Adding more RAM could reduce the need to use virtual memory.
  • Accept any other valid point.
