1.3.7 Network protocols questions

Answer each question, then reveal the hidden mark scheme underneath to check your response.

1) Explain what is meant by a communication protocol. (3 marks)

Mark scheme

1 mark for each correct point:

  • A protocol is a set of rules.
  • It controls how data is transferred.
  • It allows devices or software to communicate correctly.
  • Different protocols are used for different purposes.
  • Accept any other valid point.

2) Explain the purpose of TCP/IP. (3 marks)

Mark scheme

1 mark for each correct point:

  • TCP/IP is used for communication across networks.
  • IP helps route data to the correct destination.
  • TCP helps split data into packets and reassemble them.
  • TCP/IP is used as a basis for Internet communication.
  • Accept any other valid point.

3) Compare HTTP and HTTPS. (4 marks)

Mark scheme

1 mark for each correct point:

  • HTTP is used to transfer web pages or web data.
  • HTTPS is used to transfer web pages or web data securely.
  • HTTPS uses encryption.
  • HTTP does not provide the same level of security.
  • Both are used when accessing websites.
  • Accept any other valid point.

4) Explain the purpose of FTP. (2 marks)

Mark scheme

1 mark for each correct point:

  • FTP is used to transfer files.
  • It can upload files to a server.
  • It can download files from a server.
  • Accept any other valid point.

5) Compare POP and IMAP. (4 marks)

Mark scheme

1 mark for each correct point:

  • POP is used to retrieve or download emails from a mail server.
  • IMAP is used to access or synchronise emails from a mail server.
  • POP usually downloads emails to a device.
  • IMAP keeps emails on the server and synchronises across devices.
  • Both are email retrieval protocols.
  • Accept any other valid point.

6) Explain the purpose of SMTP. (2 marks)

Mark scheme

1 mark for each correct point:

  • SMTP is used to send emails.
  • It transfers outgoing email from a client to a mail server.
  • It can transfer email between mail servers.
  • Accept any other valid point.

7) A user enters payment details on a website. Explain why HTTPS should be used instead of HTTP. (3 marks)

Mark scheme

1 mark for each correct point:

  • HTTPS provides a secure connection.
  • HTTPS uses encryption.
  • Payment details are harder to read if intercepted.
  • It reduces the risk of sensitive data being stolen.
  • Accept any other valid point.
