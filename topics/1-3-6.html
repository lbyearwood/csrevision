1.3.6 Wired vs Wireless networks and performance factors questions

Answer each question, then reveal the hidden mark scheme underneath to check your response.

1) Compare wired and wireless network connections. (4 marks)

Mark scheme

1 mark for each correct point:

  • Wired connections use physical cables.
  • Wireless connections transmit data without cables.
  • Ethernet is an example of a wired connection.
  • Wi-Fi and Bluetooth are examples of wireless connections.
  • Wired connections are often more stable.
  • Wireless connections allow users to move around more easily.
  • Accept any other valid point.

2) Explain one advantage of a wired connection. (2 marks)

Mark scheme

1 mark for each correct point:

  • It is usually reliable or stable.
  • It is less affected by interference.
  • It can provide fast data transfer.
  • It can be more secure because physical access is needed.
  • Accept any other valid point.

3) Explain one disadvantage of a wired connection. (2 marks)

Mark scheme

1 mark for each correct point:

  • Devices need to be connected by cables.

  • Cables can create trip hazards.

Mark scheme

1 mark for each correct point:

Mark scheme

  • Accept any other valid point.

Mark scheme

  • Accept any other valid point.
