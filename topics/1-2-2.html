1.2.2 Virtual memory

Answer each question, then reveal the hidden mark scheme underneath to check your response.

1) Define what is meant by virtual memory. (2 marks)

Mark scheme

1 mark for each correct point:

  • Virtual memory is part of secondary storage used as if it were RAM.
  • It is used when there is not enough RAM available.
  • Data or instructions not currently needed are moved between RAM and secondary storage.
  • Accept any other valid point.

2) A student edits a video while also running a browser and presentation software. Explain why the computer may need to use virtual memory. (3 marks)

Mark scheme

1 mark for each correct point:

  • The computer may not have enough RAM for all running programs and data.
  • The video editor and other open programs may require more memory than the available RAM.
  • Virtual memory provides extra temporary space using secondary storage.
  • It allows programs to continue running when RAM is full.
  • It can help with multitasking when several applications are open.
  • Accept any other valid point.

3) Describe how virtual memory is used when RAM is full. (4 marks)

Mark scheme

1 mark for each correct point:

  • The operating system manages virtual memory.
  • Data or instructions not currently being used are moved out of RAM.
  • They are stored temporarily on secondary storage.
  • When needed again, they are moved back into RAM.
  • This creates space in RAM for data or instructions currently required.
  • Accept any other valid point.

4) A laptop becomes slow when switching between several large applications. Explain why using virtual memory can make the laptop run more slowly. (4 marks)

Mark scheme

1 mark for each correct point:

  • Virtual memory uses secondary storage.
  • The laptop may need to move parts of the large applications between RAM and secondary storage.
  • Secondary storage is slower to access than RAM.
  • Moving data between RAM and secondary storage takes time.
  • The CPU may have to wait for data or instructions to be loaded back into RAM.
  • Heavy use of virtual memory can reduce system performance.
  • Accept any other valid point.

5) A student has many applications open at the same time and the computer becomes slow. Explain how virtual memory may be involved. (4 marks)

Mark scheme

1 mark for each correct point:

  • Each open application uses RAM.
  • RAM may become full when many applications are open.
  • The operating system may start using virtual memory.
  • Data must be moved between RAM and secondary storage.
  • This can slow the computer because secondary storage is slower than RAM.
  • Accept any other valid point.

6) Compare RAM and virtual memory. (4 marks)

Mark scheme

1 mark for each correct point:

  • RAM is primary storage, whereas virtual memory uses secondary storage.
  • RAM is faster to access than virtual memory.
  • Both can temporarily hold data and instructions needed by running programs.
  • Virtual memory is used when RAM does not have enough space.
  • RAM is directly accessed by the CPU more quickly than data held in virtual memory.
  • Accept any other valid point.

7) Explain one advantage and one disadvantage of using virtual memory. (4 marks)

Mark scheme

Award up to 4 marks:

  • Advantage: programs can continue running when RAM is full.
  • Advantage: it can allow more programs to be open at the same time.
  • Disadvantage: it is slower than using RAM.
  • Disadvantage: heavy use of virtual memory can reduce performance.
  • Award marks for developed explanations linked to virtual memory.
  • Accept any other valid point.

8) A computer frequently relies on virtual memory. Explain why adding more RAM may improve performance. (4 marks)

Mark scheme

1 mark for each correct point:

  • More RAM gives the computer more fast primary storage.
  • More data and instructions can be kept in RAM at the same time.
  • The operating system may need to use virtual memory less often.
  • Less data needs to be swapped between RAM and secondary storage.
  • This can improve performance because RAM is faster than secondary storage.
  • Accept any other valid point.
