2.2.1 Programming fundamentals questions

Practise core programming ideas including input, output, variables, constants, operators, sequence, selection and iteration.

1) Describe the purpose of variables, constants and assignment in a program. (3 marks)

Mark scheme

Award up to 3 marks:

  • A variable stores a value that may change while the program runs.
  • A constant stores a value that should not change while the program runs.
  • Assignment gives a value to a variable or constant.
  • Accept any other valid point.

2) A shop till program inputs the prices of two items. Write code that inputs the two prices, adds them together and outputs the total. You may use OCR Exam Reference Language or Python. (4 marks)

Mark scheme

Award up to 4 marks:

  • Inputs the first number.
  • Inputs the second number.
  • Adds the two values and stores or uses the total.
  • Outputs the total.
  • Accept any other valid point.

OCR-style pseudocode example:

num1 = INPUT
num2 = INPUT
total = num1 + num2
OUTPUT total

Python example:

num1 = int(input())
num2 = int(input())
total = num1 + num2
print(total)

3) A cinema booking program checks a customer's age. Complete code that outputs Adult if age is 18 or over, otherwise outputs Child. You may use OCR Exam Reference Language or Python. (4 marks)

Mark scheme

Award up to 4 marks:

  • Uses a condition to test age.
  • Correctly uses age >= 18.
  • Outputs Adult when the condition is true.
  • Outputs Child when the condition is false.
  • Accept any other valid point.

OCR-style pseudocode example:

IF age >= 18 THEN
    OUTPUT "Adult"
ELSE
    OUTPUT "Child"
ENDIF

Python example:

if age >= 18:
    print("Adult")
else:
    print("Child")

4) A fitness app displays challenge days numbered 1 to 5. Write code that uses a count-controlled loop to output the numbers 1 to 5. You may use OCR Exam Reference Language or Python. (3 marks)

Mark scheme

Award up to 3 marks:

  • Uses a count-controlled loop.
  • Outputs each number from 1 to 5.
  • Stops after 5 has been output.
  • Accept any other valid point.

OCR-style pseudocode example:

FOR count = 1 TO 5
    OUTPUT count
NEXT count

Python example:

for count in range(1, 6):
    print(count)

5) Compare count-controlled iteration and condition-controlled iteration. (4 marks)

Mark scheme

Award up to 4 marks:

  • Count-controlled iteration repeats a known number of times.
  • Condition-controlled iteration repeats while or until a condition is met.
  • A FOR loop is often used for count-controlled iteration.
  • A WHILE loop is often used for condition-controlled iteration.
  • Accept any other valid point.
