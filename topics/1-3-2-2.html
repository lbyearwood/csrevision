1.3.2.2 Protocols

Answer each question, then reveal the hidden mark scheme underneath to check your response.

1) Define what is meant by a protocol. (2 marks)

Mark scheme

1 mark for each correct point:

  • A protocol is a set of rules.
  • Protocols control how data is transmitted between devices.
  • Protocols allow devices and systems to communicate correctly.
  • Accept any other valid point.

2) A school network uses laptops, tablets and servers made by different companies. Explain why standards and protocols are needed in this network. (3 marks)

Mark scheme

1 mark for each correct point:

  • They allow devices from different manufacturers to communicate.
  • They define agreed rules for sending and receiving data.
  • They reduce errors or incompatibility between systems.
  • They help ensure data is transmitted in a predictable way.
  • Accept any other valid point.

3) Explain the purpose of TCP/IP. (4 marks)

Mark scheme

1 mark for each correct point:

  • TCP/IP is a suite of protocols used for communication across networks.
  • IP is used for addressing and routing packets to their destination.
  • TCP helps split data into packets and reassemble them.
  • TCP helps ensure packets are delivered reliably and in the correct order.
  • TCP/IP is used on the Internet.
  • Accept any other valid point.

4) Explain the purpose of HTTP and HTTPS. (4 marks)

Mark scheme

1 mark for each correct point:

  • HTTP is used to transfer web pages/data between web servers and browsers.
  • HTTPS is the secure version of HTTP.
  • HTTPS encrypts data sent between the browser and web server.
  • HTTPS helps protect sensitive data such as passwords or payment details.
  • Both are used when accessing websites.
  • Accept any other valid point.

5) Compare HTTP and HTTPS. (4 marks)

Mark scheme

1 mark for each correct point:

  • Both are protocols used for web communication.
  • HTTP transfers web page data between browsers and web servers.
  • HTTPS does the same but adds security through encryption.
  • HTTPS is more suitable for sensitive data such as logins or payments.
  • HTTP is less secure because data is not encrypted in the same way.
  • Accept any other valid point.

6) A web developer needs to upload website files to a hosting server. Explain the purpose of FTP in this situation. (3 marks)

Mark scheme

1 mark for each correct point:

  • FTP stands for File Transfer Protocol.
  • FTP is used to transfer files between computers over a network.
  • It can be used to upload or download files to and from a server.
  • It may be used by website developers to upload files to a web server.
  • Accept any other valid point.

7) A school system sends an email to parents when a report is ready. Explain the purpose of SMTP in this situation. (3 marks)

Mark scheme

1 mark for each correct point:

  • SMTP stands for Simple Mail Transfer Protocol.
  • SMTP is used to send email.
  • It transfers outgoing email from a client to a mail server or between mail servers.
  • Accept any other valid point.

8) Compare POP and IMAP. (4 marks)

Mark scheme

1 mark for each correct point:

  • Both are used to retrieve/access email from a mail server.
  • POP downloads email from the server to the client/device.
  • POP may remove the email from the server after download, depending on settings.
  • IMAP keeps email on the server and synchronises changes across devices.
  • IMAP is useful when email is accessed from multiple devices.
  • Accept any other valid point.

9) A user wants to access the same email account from a phone, tablet and laptop, with messages kept in sync. Recommend POP or IMAP and justify your answer. (4 marks)

Mark scheme

1 mark for each correct point:

  • IMAP would be suitable.
  • IMAP stores emails on the mail server.
  • It synchronises messages across multiple devices.
  • Actions such as reading or deleting an email can be reflected on other devices.
  • POP is less suitable because it is mainly designed to download email to one device.
  • Accept any other valid point.

10) A business sends customers a password reset link through email. Explain which protocol is used to send the email and why security is still important. (4 marks)

Mark scheme

1 mark for each correct point:

  • SMTP is used to send email.
  • The email may contain sensitive information or account access links.
  • Security is important to prevent unauthorised access to accounts.
  • HTTPS should be used when the customer opens the password reset page.
  • The business should protect email accounts and servers from unauthorised access.
  • Accept any other valid point.

11) Explain the purpose of DNS. (3 marks)

Mark scheme

1 mark for each correct point:

  • DNS stands for Domain Name System.
  • DNS translates domain names into IP addresses.
  • This allows users to enter website names instead of remembering IP addresses.
  • DNS helps browsers locate the correct web server.
  • Accept any other valid point.
