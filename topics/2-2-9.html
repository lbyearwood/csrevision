2.2.9 Random number generation questions

Practise using random numbers in programs and applying them to simple GCSE scenarios.

1) Describe one reason why a program may use random number generation. (2 marks)

Mark scheme

Award up to 2 marks:

  • It can make outcomes less predictable.
  • It can be used in games, simulations or random selection.
  • Accept any other valid point.

2) A board game app needs to simulate rolling a six-sided die. Write code that generates a random whole number from 1 to 6 and outputs it. You may use OCR Exam Reference Language or Python. (3 marks)

Mark scheme

Award up to 3 marks:

  • Generates a random integer.
  • Uses the range 1 to 6 inclusive.
  • Outputs the random number.
  • Accept any other valid point.

OCR-style pseudocode example:

dice = RANDOM(1, 6)
OUTPUT dice

Python example:

import random
dice = random.randint(1, 6)
print(dice)

3) A game uses a coin toss to decide which player starts. Write code that simulates tossing a coin by generating either Heads or Tails. You may use OCR Exam Reference Language or Python. (4 marks)

Mark scheme

Award up to 4 marks:

  • Generates a random value.
  • Uses selection or random choice to choose between two outcomes.
  • Outputs Heads for one outcome.
  • Outputs Tails for the other outcome.
  • Accept any other valid point.

OCR-style pseudocode example:

choice = RANDOM(1, 2)
IF choice = 1 THEN
    OUTPUT "Heads"
ELSE
    OUTPUT "Tails"
ENDIF

Python example:

import random
choice = random.randint(1, 2)
if choice == 1:
    print("Heads")
else:
    print("Tails")

4) Explain why a quiz program might randomise the order of questions. (3 marks)

Mark scheme

Award up to 3 marks:

  • It can make the quiz different each time.
  • It can reduce the chance of users memorising the order.
  • It can make the quiz fairer or more challenging.
  • Accept any other valid point.

5) A teacher wants a program to choose a random student from a list called names. Complete code that generates a random index to select one item from the list. You may use OCR Exam Reference Language or Python. (4 marks)

Mark scheme

Award up to 4 marks:

  • Generates a random index in the valid range.
  • Uses the index to access an item in names.
  • Stores or outputs the selected item.
  • Avoids indexes outside the list.
  • Accept any other valid point.

OCR-style pseudocode example:

index = RANDOM(0, LENGTH(names) - 1)
selectedName = names[index]
OUTPUT selectedName

Python example:

import random
index = random.randint(0, len(names) - 1)
selected_name = names[index]
print(selected_name)
