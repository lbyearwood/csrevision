Paper 1

1.3.4 Network hardware

Answer each question, then reveal the hidden mark scheme underneath to check your response.

1) Explain the purpose of a Network Interface Controller/Card (NIC). (3 marks)

Mark scheme

1 mark for each correct point:

  • A NIC allows a device to connect to a network.
  • It can provide wired or wireless network access.
  • It has a MAC address.
  • It enables data to be sent and received by the device.
  • Accept any other valid point.

2) Explain the purpose of a switch in a network. (3 marks)

Mark scheme

1 mark for each correct point:

  • A switch connects devices together on a LAN.
  • It receives data from one device and forwards it to the correct device.
  • It uses MAC addresses to identify devices.
  • It reduces unnecessary data being sent to every device.
  • Accept any other valid point.

3) Explain the purpose of a router. (3 marks)

Mark scheme

1 mark for each correct point:

  • A router connects networks together.
  • It can connect a LAN to the Internet.
  • It forwards data packets between networks.
  • It uses IP addresses to help send data to the correct network.
  • Accept any other valid point.

4) Explain the purpose of a wireless access point. (3 marks)

Mark scheme

1 mark for each correct point:

  • A wireless access point allows wireless devices to connect to a network.
  • It uses Wi-Fi.
  • It connects wireless devices to a wired network.
  • It can extend wireless network coverage.
  • Accept any other valid point.

5) Explain what is meant by transmission media. (2 marks)

Mark scheme

1 mark for each correct point:

  • Transmission media is what carries data between devices.
  • It can be wired or wireless.
  • Examples include copper cables, fibre optic cables or wireless signals.
  • Accept any other valid point.

6) A school is connecting desktop computers in one computer room. Identify two pieces of network hardware that may be needed and explain their purpose. (4 marks)

Mark scheme

1 mark for each correct point:

  • A switch may be needed to connect the computers together.
  • A router may be needed to connect the LAN to another network or the Internet.
  • NICs may be needed so each computer can connect to the network.
  • Transmission media such as Ethernet cables may be needed to carry data.
  • A valid answer must identify hardware and explain its purpose.
  • Accept any other valid point.

7) Compare a switch and a router. (4 marks)

Mark scheme

1 mark for each correct point:

  • A switch connects devices on a LAN.
  • A router connects different networks together.
  • A switch can forward data to the correct device on the local network.
  • A router can forward data between networks or to the Internet.
  • Switches commonly use MAC addresses.
  • Routers commonly use IP addresses.
  • Accept any other valid point.

8) Explain why a wireless access point would be useful in a school. (3 marks)

Mark scheme

1 mark for each correct point:

  • It allows wireless devices to connect to the network.
  • It supports devices such as laptops, tablets or phones.
  • Users can connect without Ethernet cables.
  • It can provide network access in classrooms or shared spaces.
  • Accept any other valid point.
